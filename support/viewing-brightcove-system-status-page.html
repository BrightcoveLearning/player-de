Brightcove überwacht kontinuierlich den Status aller Brightcove Services und Anwendungen. Wenn es zu Betriebsunterbrechungen kommt, wird eine Notiz auf der Seite „Systemstatus“ veröffentlicht. Alle geplanten Wartungshinweise werden ebenfalls auf dieser Seite veröffentlicht.

Um auf die Systemstatusseite zuzugreifen, besuchen Sie https://status.brightcove.com oder klicken Sie in der Navigationskopfzeile auf das Support-Symbol () and then  System Status.

Alle geposteten Benachrichtigungen enthalten die folgenden Informationen:

  • Status des Vorfalls
    • Operativ (grün)
    • Benachrichtigung (Blau - Systeme sind derzeit in Betrieb, aber von Verzögerungen oder intermittierenden Ausfällen betroffen)
    • Teilweise Betriebsunterbrechung (Gelb)
    • Full-Service-Unterbrechung (Rot)
  • Komponenten - Service oder Bereich betroffen
  • Standorte
    • APAC
    • Amerika
    • EMEA
    • Japan
  • Aktueller Status und Status. Zu den Staaten gehören:
    • Untersuchung - Ingenieure untersuchen das Problem
    • Identifiziert - Ingenieure haben das Problem identifiziert und wollen das Problem korrigieren
    • Überwachung - Problem behoben und Engineering-Überwachung, um sicherzustellen, dass alles gesund ist
    • Behoben - Problem behoben

Alle geposteten Nachrichten werden auf Englisch sein und die verwendete Zeit ist Eastern Standard Time (GMT-5).

Anzeigen des Systemstatusverlaufs

Um den Systemstatusverlauf anzuzeigen, klicken Sie auf Geschichte Link am Ende der Seite.

Abonnieren von Systemstatusänderungen

Sie können sich anmelden, um benachrichtigt zu werden, wenn sich der Systemstatus ändert. Dieser Abschnitt ist wie folgt organisiert:

  • E-Mail abonnieren
  • Webhook abonnieren
  • Verwalten Sie Ihr Abonnement

E-Mail abonnieren

Um Benachrichtigungen per E-Mail zu erhalten, klicken Sie auf ABONNIEREN , geben Sie Ihre E-Mail-Adresse ein und klicken Sie dann auf Abonnieren.

Webhook abonnieren

Ein Webhook ist ein definierter Rückruf mit HTTP POST. Sie geben eine URL an, unter der der Callback ein JSON-Objekt veröffentlicht, wenn eine Statusaktualisierung stattfindet.

Um sich anzumelden, um benachrichtigt zu werden, wenn sich der Systemstatus ändert, wählen Sie die ABONNIEREN Taste. Geben Sie dann eine URL ein, um die JSON-Nutzlast zusammen mit Ihrer E-Mail-Adresse zu erhalten.

Verwalten Sie Ihr Abonnement

Nachdem Sie sich angemeldet haben, erhalten Sie eine E-Mail mit einem Link zu einer Seite „Abonnement verwalten“, auf der Sie Ihr Abonnement verwalten können. Sie können die Komponenten/Produkte/Regionen auswählen, für die Sie Benachrichtigungen erhalten möchten. Treffen Sie Ihre Auswahl und klicken Sie dann auf Abonnement speichern.

Um sich abzumelden, klicken Sie auf das Abbestellen Link unten auf der Seite Abonnement verwalten.

Benachrichtigungen in Slack abrufen

Slack hat ein eigenes Webhook-System, das zum Verbinden von Statusaktualisierungen verwendet werden kann. Weitere Informationen finden Sie in der Slack-Dokumentation.
