Grundlagen für Spielerentwickler: Erweiterter Spielercode

In diesem Thema erfahren Sie, wie Sie ein Video programmgesteuert automatisch abspielen.
 

Schritte

  1. Beginnen Sie mit dem Standard-Best-Practice-Code für die programmgesteuerte Steuerung der Spieler. 
    <video id="myPlayerID"
  data-video-id="5785633544001"
  data-account="921483702001"
  data-player="Uj7Yz80yM"
  data-embed="default"
  data-application-id
  class="video-js"
  controls
  width="640"
  height="360"></video>
<script src="//players.brightcove.net/921483702001/Uj7Yz80yM_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;

  });
</script>
  2. In der anonymen Funktion für die videojs.getPlayer() Methode, die der Variablenzuweisung folgt, schaltet den Player stumm, damit er automatisch wiedergegeben werden kann. 
    myPlayer.muted(true);
  3. Verwenden Sie die des Spielers on() Methode zum Hinzufügen einer Ereignishandlerfunktion für die loadedmetadata Veranstaltung. 
    myPlayer.on('loadedmetadata', function(){

});
  4. Spielen Sie im Event-Handler das Video ab. 
    myPlayer.play()
