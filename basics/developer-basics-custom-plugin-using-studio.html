Grundlagen für Spielerentwickler: Benutzerdefiniertes Plugin - Verwenden von Studio

In diesem Thema erfahren Sie, wie Sie einem Brightcove Player mit Studio ein benutzerdefiniertes Plugin zuweisen.
 

Schritte

  1. Wählen Sie in Studio das PLAYERS Modul aus.
  2. Klicken Sie auf den Player, zu dem Sie das benutzerdefinierte Plugin hinzufügen möchten.
  3. Wählen Sie in den Player-Eigenschaften auf der linken Seite die Option Plugins Möglichkeit .
  4. Wählen Sie im Dropdown-Menü Plugin hinzufügen die Option Benutzerdefiniertes Plugin aus.
  5. Fügen Sie Folgendes hinzu:
    • Name des Plugins
    • JavaScript URL
    • CSS-URL - falls erforderlich
    • Optionen (JSON) - falls erforderlich
  6. Speichern Sie den Player.
  7. Veröffentlichen der Spieler.
