Grundlagen für Spielerentwickler: Vorbereitung zum Schreiben von Code

In diesem Thema erfahren Sie, wie Sie eine Standardvorlage für bewährte Verfahren erstellen, die Sie darauf vorbereitet, Code zur Steuerung des Players zu schreiben.

Schritte

  1. Fügen Sie eine Instanz des Player-Codes für die erweiterte Implementierung in eine HTML-Seite ein. 
    <video data-video-id="5831704295001"
  data-account="921483702001"
  data-player="Uj7Yz80yM"
  data-embed="default"
  data-application-id
  class="video-js"
  controls
  width="640"
  height="360"></video>
<script src="//players.brightcove.net/921483702001/Uj7Yz80yM_default/index.min.js"></script>
  2. Fügen Sie eine hinzu id Attribut auf die <video> tag und weisen Sie ihm einen Wert von zu myPlayerID. 
    <video id="myPlayerID"
  3. Füge hinzu ein <script> Block direkt über dem </body> Etikett.
  4. In dem script Block, verwenden Sie die videojs.getPlayer() Methode, um einen Verweis auf den Player auf der Seite zu erhalten. Verwenden Sie die id früher als Argument der Methode hinzugefügt. 
    <script>
  videojs.getPlayer('myPlayerID')
</script>
  5. Verwenden Sie die ready() Methode und codieren Sie eine anonyme Rückruffunktion als Argument. 
    <script>
  videojs.getPlayer('myPlayerID').ready(function() {

  )};
</script>
  6. Deklarieren Sie eine Variable mit dem Namen myPlayer in der anonymen Funktion und weisen Sie ihm das Schlüsselwort zu this , der den Spieler im Kontext der anonymen Funktion darstellt. 
    var myPlayer = this;

Code abgeschlossen

  )};
