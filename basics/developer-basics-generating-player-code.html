Grundlagen für Spielerentwickler: Spielercode generieren

In diesem Thema erfahren Sie, wie Sie Player-Code generieren, den Sie später in der Player-Entwicklung verwenden können.
  1. Öffnen Sie in Studio die SPIELER Modul.
  2. Drücke den Neuer Spieler Taste.
  3. Geben Sie den Spieler und den Namen ein und klicken Sie dann auf speichern.
  4. Öffnen Sie das MEDIA-Modul.
  5. Klicken Sie auf ein Video, um es zu veröffentlichen.
  6. Klicken Sie auf der Seite mit den Player-Eigenschaften auf Veröffentlichen und einbetten Schaltfläche, die aus der Dropdown-Liste auswählen Internet Spieler .
  7. Wählen Sie für den tatsächlichen Spieler den gerade erstellten Spieler aus.
  8. In dem Code kopieren Abschnitt wählen Sie die Fortgeschrittene Implementierung.
  9. Stellen Sie Folgendes ein:
    • Größenbestimmung: Fest
    • Seitenverhältnis: Passen Sie Ihr Video an
    • Spielerbreite:: Gewünschte Breite
  10. Zuletzt kopieren Sie den Code.
