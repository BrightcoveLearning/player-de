    Verwenden von Live-RTMP-Ausgaben zum Streamen auf Facebook und YouTube

    In diesem Thema erfahren Sie, wie Sie RTMP-Ausgaben im Live-Modul verwenden, um Live-Events auf Facebook und YouTube zu streamen.

    Einführung

    Das Live-Modul kann verwendet werden, um ein Live-Event sowohl auf Desktop- als auch auf Mobilgeräten zu übertragen. Nach Eingabe der Veranstaltungsinformationen ins Live-Modul erscheinen verschiedene Codierungseinstellungen, die Sie in Ihren Codierer integrieren können. In der Regel wird der Livestream innerhalb eines Brightcove Players veröffentlicht. Die Seite Kontrollraum des Live-Moduls bietet die Möglichkeit, zusätzliche RTMP-Ausgabe-URLs einzugeben. Die RTMP-Ausgabeeinstellungen können verwendet werden, um den Livestream an Social-Media-Seiten zu senden. Beachten Sie, dass die RTMP-Ausgabestunden für die Veranstaltungszeiten in Rechnung gestellt werden. In diesem Thema werden die Schritte zum Erstellen eines einfachen Live-Events behandelt und dann gezeigt, wie die RTMP-Ausgaben verwendet werden können, um ein Live-Event auf Facebook und YouTube zu streamen.

    Hinweis: Anzeigen, einschließlich serverseitiger Werbung (SSAI), werden nicht zu RTMP-Ausgaben gestreamt. Stattdessen wird der werbefreie Stream verwendet.

    Erstellen einer neuen Veranstaltung

    Führen Sie die folgenden Schritte aus, um eine neue Veranstaltung zu erstellen:

    1. Öffnen Sie das Live-Modul.
    2. Klicken Sie auf der Seite „Ereignisse verwalten“ auf die Schaltfläche „ Veranstaltung erstellen “ oder klicken Sie im linken Navigationsbereich auf Live-Event erstellen oder Live-Channel erstellen. Die Seite „Live-Event erstellen oder Live-Channel erstellen“ wird geöffnet.
    3. Geben Sie den Veranstaltungsnamen und alle erforderlichen benutzerdefinierten Felder ein.
    4. Wählen Sie ein Live-Ingest-Profilaus.
    5. Wählen Sie Event nach Abschluss des Ereignisses in Video-Asset konvertieren aus, um eine VOD-Version des Live-Videos in Ihrem Video Cloud-Konto zu speichern.
    6. Klicken Sie auf ERWEITERTE OPTIONEN und konfigurieren Sie die erweiterten Optionen nach Bedarf.
    7. Klicken Sie auf Streaming starten, um das Live-Event zu starten. Die Control Room-Seite wird mit Details zum Live-Ereignis geöffnet, einschließlich des Streaming-Endpunkts (RTMP-URL) und des Stream-Namens , die von Ihrem Encoder verwendet werden.
      Hinweis: Die Verarbeitung Ihrer Streams dauert ungefähr eine Minute.
      Hinweis: Standardmäßig muss der Encoder innerhalb von 30 Minuten nach der Erstellung eines neuen Ereignisses gestartet werden.
    8. Beginnen Sie mit dem Streaming vom Encoder und der Live-Stream wird auf der Control Room-Seite angezeigt.

    Sobald das Live-Event begonnen hat, kann der Live-Stream an Facebook und/oder YouTubegesendet werden.

    Hinweis: Stellen Sie sicher, dass der Encoder geschlossen/gestoppt ist, wenn der Live-Stream abgeschlossen ist. Bleibt die Verbindung zwischen einem Codierer und Brightcove bestehen, fallen u. U. unerwartete Verbindungsgebühren an, auch wenn keine Daten übertragen werden.

    Streaming auf Facebook Live

    Facebook Live ist eine  Live-Videostreaming-Funktion auf Facebook , mit der Sie über Ihre Unternehmensseite oder Ihr persönliches Profil ein  Live-Event an Ihr Publikum übertragen können. Live-Videos werden im Newsfeed einer Person während der Live-Veranstaltung sowie nach dem Ende der Veranstaltung angezeigt.

    Hinweis: Vollständige Details zum Facebook-Livestreaming finden Sie in der Facebook-Dokumentation.

    Sobald das Live-Streaming im Live-Modul gestartet wurde, befolgen Sie diese Schritte, um den Live-Stream in Facebook zu verwenden.

    1. Melden Sie sich an Ihrem Facebook-Konto an.
    2. Erstellen Sie einen neuen Beitrag und klicken Sie auf die Option Live-Video .
    3. Facebook fordert möglicherweise den Zugriff auf Ihre Computerkamera und Ihr Mikrofon auf. In diesem Thema wird eine externe DSLR-Kamera für das Ereignis verwendet. Klicken Sie daher auf Connect.
    4. Facebook zeigt eine Server-URL und einen Stream-Keyan. Diese Werte werden im Live-Modul verwendet.
    5. Kopieren Sie die Server-URL in die Zwischenablage.
    6. Fügen Sie im Live-Modul die Server-URL in das Feld RTMP-Ausgabe-URL auf der Control Room-Seite ein.
    7. Kopiere den Stream Key von Facebook in die Zwischenablage.
    8. Fügen Sie im Live-Modul den Streamschlüssel am Ende des Feldes RTMP-Ausgabe-URL ein. Das Feld RTMP-Ausgabe-URL ist die URL des Facebook-Servers mit angehängtem Streamschlüssel , alles als ein Wert.
      Hinweis: Das einzige derzeit für RTMP-URLs unterstützte Format ist rtmps://host:port/application/streamName.
    9. Wählen Sie eine Formatvariante aus der Dropdown-Liste „Formatvariante auswählen“ aus und klicken Sie auf „RTMP-
    10. Vergewissern Sie sich unter dem RTMP-Abschnitt, dass die Verbindung erfolgreich war.
    11. Bestätige, dass der Livestream in Facebook erscheint. Geben Sie einen Titel und eine Beschreibung für Facebook ein.
    12. Klicken Sie auf „ Go Live

    Der Livestream wird in Facebook beginnen.

    Klicken Sie auf Beenden , um den Facebook-Livestre Denken Sie daran, den Encoder zu stoppen, wenn das Ereignis abgeschlossen ist.

    Streaming auf YouTube

    YouTube bietet die Möglichkeit, Live-Streaming-Videos an Ihre Zielgruppe zu liefern. Bevor du zum ersten Mal einen Live-Stream beginnst, musst du deinen YouTube-Kanal aktivieren.

    Hinweis: Die erstmalige Aktivierung eines Live-Streams kann bis zu 24 Stunden dauern. Nach der Aktivierung kann Ihr Stream sofort live geschaltet werden.
    Hinweis: Vollständige Details zum YouTube-Live-Streaming findest du in der YouTube-Dokumentation.

    Sobald das Live-Streaming im Live-Modul gestartet wurde, befolgen Sie diese Schritte, um den Live-Stream in YouTube zu verwenden.

    1. Melde dich bei deinem YouTube-Konto an
    2. Rufen Sie die YouTube Studio-Seite auf https://studio.youtube.com auf.
    3. Klicken Sie in der linken Navigationsleiste auf Andere Funktionen > Live-Stream jetzt.
    4. YouTube fordert möglicherweise den Zugriff auf Ihre Computerkamera und Ihr Mikrofon auf. In diesem Thema wird eine externe DSLR-Kamera für das Ereignis verwendet. Klicken Sie daher auf Connect.
    5. YouTube zeigt eine Server-URL und einen Stream-Namen/-Schlüsselan. Diese Werte werden im Live-Modul verwendet.
    6. Kopieren Sie die Server-URL in die Zwischenablage.
    7. Fügen Sie im Live-Modul die Server-URL in das Feld RTMP-Ausgabe-URL auf der Control Room-Seite ein.
      Hinweis: Das einzige derzeit für RTMP-URLs unterstützte Format ist rtmps://host:port/application/streamName.
    8. Kopiere den Stream-Namen/Schlüssel von YouTube in die Zwischenablage.
    9. Fügen Sie im Live-Modul den Stream-Namen/-Schlüssel am Ende des Felds RTMP-Ausgabe-URL ein. Das Feld RTMP-Ausgabe-URL ist die YouTube-Server-URL mit angehängtem Stream-Name/Schlüssel , alles als ein Wert.
    10. Wählen Sie eine Formatvariante aus der Dropdown-Liste „Formatvariante auswählen“ aus und klicken Sie auf „RTMP-
    11. Vergewissern Sie sich unter dem RTMP-Abschnitt, dass die Verbindung erfolgreich war.
    12. Bestätige, dass der Livestream auf YouTube erscheint

    Der Livestream beginnt auf YouTube.

    Klicken Sie auf Return to YouTube Studio und stoppen Sie dann den Encoder, um den Livestream zu beenden. Wenn dein Stream abgeschlossen ist, wird automatisch ein öffentliches Video erstellt und hochgeladen, damit die Leute sie später ansehen können.
