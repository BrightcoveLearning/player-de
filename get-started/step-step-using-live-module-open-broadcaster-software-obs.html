    Schritt für Schritt: Verwenden des Live-Moduls mit Open Broadcaster Software (OBS)

    Der Zweck dieses Quick Start besteht darin, Ihnen zu zeigen, wie Sie das Video Cloud Live-Modul und Open Broadcaster Software (OBS) zur Übertragung einer Live-Veranstaltung verwenden.
     

    Informationen zu allen im Live-Modul verfügbaren Funktionen und Funktionen finden Sie unter Erstellen und Verwalten von Live-Events mit dem Live-Modul. Richtlinien und Best Practices für das Live-Modul enthält optimale Vorgehensweisen und Empfehlungen, die eine hohe Qualität und stabiles Live-Streaming ermöglichen. Eine Liste aller von Brightcove für das Bereitstellen von Live-Streaming-Videos verfügbaren Optionen finden Sie unter Bereitstellen von Live-Streams.

    Hinweis: Eine Version dieses Dokuments, die den Telestream Wirecast-Encoder verwendet, finden Sie unter Schritt-für-Schritt: Verwenden des Live-Moduls mit Telestream Wirecast.
    Hinweis: Unterstützung für Live-Streaming steht nur für Video Cloud Pro- und Enterprise-Publisher zur Verfügung.

    Nach der Lektüre dieser Kurzanleitung sind Sie mit den folgenden Aufgaben vertraut:

    • Verwenden des Video Cloud Live-Moduls zum Erstellen einer Live-Veranstaltung
    • Konfigurieren Ihres Codierers für eine Live-Veranstaltung
    • Zuschneiden von Veranstaltungen und Speichern als VOD-Objekt in einem Video Cloud-Konto

    Dieser Quick Start hilft Ihnen, ein Live-Event mit Open Broadcaster Software (OBS) Studio auf verschiedene Geräte zu übertragen. OBS ist Open Source Software für Videoaufzeichnung und Live-Streaming und kann hier heruntergeladenwerden. Andere Codierer können ebenfalls mit dem Live-Modul verwendet werden.

    Zielgruppe

    Jeder Video Cloud-Publisher, der eine Live-Streaming-Veranstaltung mit dem Video Cloud Live-Modul übertragen muss.

    Voraussetzungen

    • Ein Video Cloud Pro- oder Enterprise-Konto mit Zugriff auf das Live-Modul
    • Eine Kamera für die Übertragung der Veranstaltung
    • Sie müssen über eine eigene Codierungssoftware verfügen. Dieser Quick Start verwendet OBS Studio , das kostenlos heruntergeladen werden kann (siehe eine Liste der unterstützten Encoder)

    Einschränkungen

    • Ihr Encoder muss einen RTMP- oder SRT-Stream ausgeben
    • Bei Live-Übertragungen entsteht eine Verzögerung von 20 bis 30 Sekunden.
    • Das Streaming muss innerhalb von 30 Minuten nach Veranstaltungsbeginn beginnen (Kontoeinstellung auf Standardebene)
    • Benutzer von Internet Explorer benötigen Version 10 oder später.

    Schritte zur Übertragung eines Live-Events

    Wenn Sie Ihre Live-Veranstaltung mithilfe des Live-Moduls streamen möchten, müssen Sie folgende Aufgaben ausführen:

    • Erstellen Sie ein neues Ereignis mit dem Video Cloud Live-Modul
    • Konfigurieren Sie Ihren Encoder
    • Veröffentlichen Sie den Livestre
    • Erstellen Sie einen Clip aus dem Live-Event

    Erstellen eines neuen Live-Events

    1. Navigieren Sie in einem Browserfenster zu Video Cloud Studio.
    2. Geben Sie Ihre Video Cloud-Anmeldeinformationen ein, und klicken Sie auf Sign In.
    3. Der aktuelle Video Cloud-Kontoname erscheint auf der Seite oben rechts. Wenn Sie mehrere Konten besitzen, klicken Sie auf die Dropdownliste zur Kontoauswahl, und wählen Sie das Konto, in dem das Live-Objekt gespeichert werden soll. Wenn das ausgewählte Konto nicht für Live aktiviert wurde, wird das Live-Modul im Testmodus geöffnet.
    4. Klicken Sie im Navigations-Header auf Live.
    5. Klicken Sie auf die Schaltfläche „ Ereignis erstellen “, um ein neues Ereignis zu erstellen.
    6. Geben Sie einen Ereignisnamenein.
    7. Geben Sie einige Tags für die Veranstaltung ein.
    8. Wenn Ihr Konto mit erforderlichen benutzerdefinierten Feldern konfiguriert wurde, stellen Sie sicher, dass den benutzerdefinierten Feldern Werte zugewiesen sind.
    9. Wählen Sie ein Live Ingest-Profil aus der Dropdown-Liste aus. Das Live-Ingest-Profil steuert die Anzahl und Qualität der erstellten Live-Darstellungen.
    10. Wählen Sie „ Ereignis in Video-Asset umwandeln“, wennes Dadurch wird das Ereignis als Video im Medienmodul gespeichert.
    11. Klicken Sie auf den Link ERWEITERTE OPTIONEN , um die erweiterten Optionen zu erweitern.
    12. Stellen Sie die Wiederverbindungszeit auf 1 Minute ein.
    13. Wählen Sie die Region aus, die Ihrem Standort am nächsten liegt.
      OBS Encoder unterstützt das SRT-Protokoll, und Sie können diese Option im Abschnitt „ Eingabeformat “ des Menüs „ Erweiterte Optionen “ auswählen.
    14. Klicken Sie auf Start Streaming (Streaming starten). Das Live-Modul gibt einen Stream-Endpoint (RTMP-URL) und einen Stream-Namenzurück. Diese Werte werden in den Codierereinstellungen verwendet.
      Wenn Sie das SRT-Protokoll für das Live-Ereignis ausgewählt haben, ist dieses in OBS verfügbar, indem Sie unter Stream settings Service „Benutzerdefiniert“ wählen und dann Ihren srt:// Pfad in das Feld Server eingeben.

    Konfigurieren Sie Ihren Encoder

    In diesem Abschnitt werden wir den OBS Studio-Encoder konfigurieren und verwenden, um ein Live-Streaming-Event zu unterstützen. Die OBS Studio-Software ist eine Desktop-Anwendung, die Eingaben von Ihrer Kamera erfasst und einen Stream erzeugt, der von einem CDN bereitgestellt werden kann. Beachten Sie, dass eventuell andere Codierungslösungen auf Hardware- und Softwarebasis verfügbar sind, die sich möglicherweise besser für die Bereitstellung Ihrer Live-Streaming-Veranstaltung eignen. Sie können die OBS Studio Software hierherunterladen.

    Hinweis: Sie sollten die OBS-Software installiert und eine Kamera an Ihren Computer angeschlossen haben, bevor Sie fortfahren
    Hinweis: In diesem Beispiel wird eine DSLR verwendet, die über den Blackmagic Design UltraStudio Mini Recorder mit einem Laptop verbunden ist.
    Hinweis: OBS bietet die Möglichkeit, eine lokale Kopie des Streams aufzuzeichnen. Wenn Sie sich dafür entscheiden, erhalten Sie eine lokale Kopie des Streams, aus der Sie Clips erstellen können, falls während der Übertragung des Ereignisses Netzwerkprobleme auftreten. Informationen zum Aufzeichnen einer lokalen Datei finden Sie in der OSP-Dokumentation.
    1. Öffnen Sie OBS Studio.
    2. Klicken Sie im Bereich „ Quellen “ auf + , um eine neue Quelle hinzuzufügen. Wählen Sie eine Quelle aus der Liste aus. Dieser Quick Start verwendet eine DSLR, die über den Blackmagic Design UltraStudio Mini Recorder mit einem Laptop verbunden ist, daher ist das Blackmagic-Gerät ausgewählt.
    3. Geben Sie einen Namen für die neue Quelle ein und klicken Sie auf OK.
    4. Wählen Sie Ihre Kamera aus der Geräteliste aus. Eine Vorschau sollte im Eigenschaftendialogfeld angezeigt werden. Klicken Sie auf OK.
    5. Klicken Sie im Abschnitt Steuerelemente auf Einstellungen.
    6. Klicken Sie links im Dialogfeld „Einstellungen“ auf Stream .
    7. Wählen Sie für den Streamtyp den benutzerdefinierten Streaming-Serveraus.
    8. Verwenden Sie für die URL die Stream-URL , die vom Live-Modul bereitgestellt wird.
    9. Verwenden Sie für den Stream-Schlüssel alive.
    10. Klicken Sie auf OK , um die Einstellungen zu speichern.
    11. Klicken Sie im Abschnitt Steuerelemente auf Streaming starten. Der Live-Stream sollte starten.
    Hinweis: Die Verarbeitung Ihrer Streams dauert ungefähr eine Minute.

    Veröffentlichen des Live-Events

    Um den Einbettungscode für das Live-Ereignis zu generieren, führen Sie die folgenden Schritte aus:

    1. Kehren Sie zum Live-Modul zurück.
    2. Wenn die Control Room-Seite für das Ereignis nicht angezeigt wird, klicken Sie auf die Registerkarte Live , um eine Liste der Live-Events anzuzeigen. Klicken Sie auf den Namen des Ereignisses, um die Control Room-Seite zu öffnen.
    3. Stellen Sie sicher, dass Sie den Live-Stream sehen. Beachten Sie, dass es nach dem Start des Encoders eine Minute dauern kann, bis der Live-Stream erscheint.
    4. Klicken Sie auf Veröffentlichen und Einbetten.
    5. Wähle einen Player aus und klicke dann auf die Player-URL , um den Livestream anzuzeigen.
    6. Wenn Sie mit dem Live-Stream fertig sind, klicken Sie im Live-Modul auf die Schaltfläche Stream stoppen . Kehren Sie außerdem zu OBS Studio zurück und klicken Sie auf die Schaltfläche „Stream“, um das Streamen des Encoders zu verhindern.
      Hinweis: Wichtig ist das Schließen/Beenden des Codierers nach Abschluss des Live-Streams. Bleibt die Verbindung zwischen einem Codierer und Brightcove bestehen, fallen u. U. unerwartete Verbindungsgebühren an, auch wenn keine Daten übertragen werden.

    Erstellen eines Clips

    Das Live-Modul bietet die Möglichkeit, Videoclips aus abgeschlossenen Ereignissen zu erstellen. Clips können auch erstellt werden, während ein Live-Event läuft. Es gibt mehrere Gründe, warum Sie einen Clip erstellen möchten:

    • Um einen kurzen Teaser-Clip zu erstellen, der in sozialen Medien gepostet werden kann
    • So begrenzen Sie die Länge eines Videos, das Sie posten möchten (die meisten Plattformen begrenzen die Länge von Videos)
    • Sie haben festgestellt, dass kürzere Videos in den sozialen Medien besser abschneiden

    Clips können nur innerhalb von 7 Tagen nach Ende des Events erstellt werden. Danach wird die Schaltfläche „ Clip erstellen “ deaktiviert. Wenn das Event länger als 24 Stunden dauert, sind nur die letzten 24 Stunden anklippbar.

    Hinweis: Ausführliche Informationen zur Verwendung der Clipping-Schnittstelle finden Sie unter Erstellen eines Videoclips mit dem Live-Modul.

    Um einen Clip zu erstellen, gehen Sie folgendermaßen vor:

    1. Kehren Sie zum Live-Modul zurück.
    2. Klicken Sie auf die Registerkarte Abgeschlossen .
    3. Suchen Sie das Ereignis und klicken Sie auf den Ereignisnamen, um den Control Room zu öffnen.
    4. Klicken Sie auf „ Clip erstellen Der Clipping-Editor wird geöffnet.
    5. Klicken und ziehen Sie die Clipping-Handles () auf der Fokus-Timeline, um die Start- und Endpunkte für den Clip festzulegen.
    6. Klicken Sie auf Clip-Vorschau , um eine Vorschau des Clips Die Start- und Endpunkte können nach Bedarf angepasst werden.
    7. Klicken Sie auf „ Clip speichern
    8. Geben Sie einen Clip-Namen und andere Metadatenwerte ein. Alle erforderlichen Felder für das Konto werden ebenfalls angezeigt und sollten Werte zugewiesen werden.
    9. Klicken Sie auf „ Clip erstellen Der Aufgabe-Prozess beginnt und der Clip wird in Ihrem Video Cloud-Konto gespeichert.
    10. Clips werden unter dem Clip-Editor angezeigt. Klicken Sie auf In Social veröffentlichen, um den Clip mit Brightcove Social an einem Ihrer sozialen Ziele zu veröffentlichen. Informationen zum Veröffentlichen von Videos mit dem Social-Modul finden Sie unter Veröffentlichen von Videos auf Social Media-Plattformen.
    11. Klicken Sie auf „ Zurück zum Kontrollraum“: Breadcrumb abgeschlossen . Alle Clips werden auch im Control Room für das Ereignis angezeigt.
      Hinweis: Klicken Sie mehrmals auf die Schaltfläche „Create Clip“, um mehrere Clips von Ihrer Übertragung zu erstellen. Auf diese Weise können Sie Videos für jede Übertragung beschneiden und beliebig oft speichern und die Clips in Ihrer Medienbibliothek als getrennte VOD-Objekte speichern.

    Fehlerbehebung bei Problemen mit dem Live-Stream

    Falls Sie beim Einrichten Ihrer Live-Veranstaltung Hilfe benötigen, kontaktieren Sie uns. Um sicherzustellen, dass Ihr Problem schnellstmöglich gelöst wird, sollten Sie die folgenden Informationen für den Support bereitstellen:

    • Beschreibung der speziellen Symptome des Streams. Beispiel: Ist überhaupt keine Wiedergabe möglich, oder kommt es zu Stockungen oder zum Einfrieren?
    • Information, ob der Stream bisher korrekt verfügbar war
    • Im Codierer verwendete Einstiegspunkt-URL
    • Verwendete Codiersoftware und -hardware
    • URL für den Player, auf dem Sie die Live-Veranstaltung veröffentlicht haben
    • Video-ID Ihrer Live-Datei in Video Cloud Studio
    • Ergebnisse einer Ablaufverfolgungsroute von Ihrem Encoder zum Host des Veröffentlichungspunkts
