Brightcove Plugins

Im Folgenden finden Sie eine Zusammenfassung der von Brightcove gelieferten Player-Plugins. Plugins werden verwendet, um den Brightcove Player zu verbessern.

Plugin Standardmäßig geladen Beschreibung
360º Video   Dieses Plugin verwandelt ein richtig aufgenommenes Videoelement in ein HTML5-Panorama-360°-Video, das im Brightcove Player abgespielt werden kann.
Nur Anzeige   Mit diesem Plugin können Sie nur IMA3-Anzeigen in einem Player abspielen. Es werden keine normalen Videoinhalte geliefert.
Werbung mit FreeWheel   Dieses Plugin ermöglicht die FreeWheel Anzeigentechnologie für Flash oder HTML5 in Brightcove Player.
Werbung mit IMA3   Integriert den Brightcove Player in Googles Interactive Media Ads (IMA) für HTML5 Version 3. Auf diese Weise können Sie VAST Anzeigen für Ihren Spieler anfordern und verfolgen.
Werbung mit SSAI   Mit der serverseitigen Anzeigeneinfügung (SSAI) können Sie Anzeigen in Ihre Videos einbetten, sodass sie nicht von Werbeblockern im Browser blockiert werden können. Standardmäßig erzwingt das Plugin, dass alle Anzeigen angesehen werden, und zeigt während der Wiedergabe einen Anzeigenzähl-Timer an. Sie können dieses Plugin ganz einfach anpassen, um Werbung zu überspringen.
Luftspiel   Ermöglicht das Streamen von Inhalten von Brightcove Player auf Ihr Apple TV.
Chromecast   Ermöglicht Brightcove Player, Videos von Ihrem Desktop oder Android Chrome-Browser auf das Chromecast-Gerät zu übertragen.
Benutzerdefinierter Endbildschirm   Zeigt den vom Herausgeber bereitgestellten HTML am Ende der Videowiedergabe an.
Miniaturansichtsvorschauen anzeigen   Zeigt an definierten Stellen im Video ein Vorschaubild über der Timeline an. Wenn sich der Benutzer über die Timeline bewegt, werden Miniaturansichten für bestimmte Zeitbereiche angezeigt.
DRM (formell DASH genannt)   Wird verwendet, um DRM in Brightcove Player zu implementieren.
Fehlermeldungen Y Ermöglicht dem Player, benutzerfreundliche Nachrichten anzuzeigen, wenn ein Fehler auftritt. Die Anzeige ist ein Overlay, das halbtransparent und durch das Standard-Stylesheet gestaltet ist.
Google-Tag-Manager   Ermöglicht die Integration mit Google Tag Manager-Funktionalität mit Brightcove Player.
HLS Y Spielt HLS-Video auf Plattformen ab, die HLS nicht unterstützen, aber Flash Player haben. Der ermöglicht es, den Videoinhalt im m3u8-Manifest im Player abzuspielen.
Overlay-Anzeige   Zeigt einfache Nachrichten als halbtransparente Overlays auf dem Player an. Die Nachrichtenanzeige kann auf Spielerereignissen wie Wiedergabe oder Pause oder zu einem bestimmten Zeitpunkt während der Videowiedergabe basieren.
Bild im Bild   Ermöglicht dem Benutzer, die Seite nach unten zu scrollen, und der Player wird an eine Ecke der Webseite angeheftet.
Playlist-Endbildschirm   Dieses Plugin ermöglicht die Anzeige einer Playlist am Ende eines Videos. Sie können dies einem Player hinzufügen, indem Sie entweder Video Cloud Studio verwenden oder Code mit Brightcove Player verwenden.
Wiedergabelisten-Benutzeroberfläche   Verbessert das Erscheinungsbild der Standard-Playlist-Funktionalität.
Auswahl der Qualität   Bietet eine Menüschaltfläche in der Steuerleiste des Players, die die manuelle Auswahl der Wiedergabequalität für HLS- oder Dash-Quellen ermöglicht.
Soziale Medien   Ermöglicht es Nutzern, ein Video auf Facebook, Twitter, Tumblr, Pinterest oder LinkedIn zu teilen.
Tealium   Ermöglicht die Integration mit dem Tealium iQ Tag-Manager.
Thumbnail-Suche   Implementiert die Suche nach Miniaturansichten mit Brightcove Player.

Open Source-Plugins

Der Open Source Video.js Player, auf dem der Brightcove Player basiert, hat viele Plugins dafür gebaut. Diese Plugins sind mit dem Brightcove Player kompatibel, solange die Version des Players Video.js dieselbe Version ist, auf der der Brightcove Player basiert. Der Brightcove Player kann versionsmäßig entweder vor oder hinter dem Open Source Video.js Player sein. Der Link zur Seite Video.js Plugins lautet https://github.com/videojs/video.js/wiki/Plugins.

Ebenfalls erhältlich sind Brightcove Player spezifische Open Source Plugins. Als Beispiele dienen Plugins für Conviva- und Nielsen-Analytics. Beachten Sie, dass Brightcove diese Plugins nicht unterstützt oder dokumentiert. Kunden sollten sich an ihre Kundenbetreuer bei Conviva/Nielsen wenden, um Einzelheiten und Zugang zu erhalten.

Erhältlich bei Moat, einem Unternehmen, das Publishern eine einfache Möglichkeit bietet, die Sichtbarkeit der wiedergegebenen Anzeigen zu messen und darüber zu berichten, ist ein Plugin, das mit Brightcove Player zusammenarbeitet. Um Zugriff auf Moat-Plugin für den Brightcove Player zu erhalten, sollten sich Kunden an ihren Moat-Kundenbetreuer wenden.
