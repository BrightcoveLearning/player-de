    Identifizieren und Löschen von Videos mit geringer Leistung

    In diesem Thema erfahren Sie, wie Sie mit Video Cloud Studio und einer Tabelle Videos mit geringer Leistung identifizieren und löschen.

    Einführung

     

    Aus einer Reihe von Gründen ist es wahrscheinlich am besten, keine Videos in Ihrer Bibliothek zu haben, die nicht angezeigt werden, und Sie vermuten, dass sie auch in Zukunft nicht mehr angezeigt werden. In diesem Dokument erfahren Sie, wie Sie diese Videos identifizieren und aus Ihrer Bibliothek entfernen. Die folgenden Aufzählungspunkte bieten einen allgemeinen Überblick über den Prozess. Die detaillierten Schritte folgen dann.

    • Verwenden Sie das Medienmodul, um eine Datei zu exportieren, die eine Reihe von Informationen für jedes Video in Ihrer Bibliothek enthält.
    • Verwenden Sie das Analytics-Modul, um einen benutzerdefinierten Bericht zu erstellen, der die Video-ID, den Titel, die Videoimpressionen und die Videoansichten enthält. Der Bericht wird in einem CSV-Format (Comma Separated Values) exportiert und listet nur die Videos mit Videoansichten und / oder Impressionen ungleich Null auf.
    • Importieren Sie beide CSV-Dateien in eine Google Text & Tabellen-Tabelle.
    • Fügen Sie der Tabelle "Alle Videos" eine Spalte hinzu, die bestimmt, ob das in einer bestimmten Zeile detaillierte Video Videoansichten enthält oder nicht. Überprüfen Sie dazu, ob eine ID in der Tabelle "Alle Videos" einen übereinstimmenden Eintrag in der Tabelle "Analytics-Bericht" enthält.
    • Kopieren Sie die IDs der Videos ohne Ansichten in ein Google Doc und bearbeiten Sie das Format, das für einen Massenlöschvorgang verwendet werden soll.
    Das Löschen von Videos ist ein drastischer Vorgang, und es ist nicht möglich, sie wiederherzustellen. Stellen Sie aus Sicherheitsgründen sicher, dass Sie eine lokale Kopie eines Videos haben, bevor Sie es löschen.

    Exportieren Sie alle Videodaten

    Von dem Medien Modul können Sie einen Bericht über alle Medien in Ihrer Bibliothek erhalten. Dies wird später in einer Tabelle verwendet.

    1. Betreten Sie Studios Medien Modul.
    2. Klick auf das Export Schaltfläche oben auf der Seite.
    3. Stellen Sie im folgenden Dialogfeld sicher, dass Alle Videos in der Liste ist ausgewählt.
    4. Drücke den Daten exportieren Taste. Beachten Sie, dass dieser Bericht Informationen für maximal 15.000 Videos exportiert.
    5. Wenn der Bericht fertig erstellt wurde, klicken Sie auf Download-Datei Taste.
    6. Beachten Sie, dass der Dateiname Ihrer Konto-ID entspricht und die Datei im CSV-Format (durch Kommas getrennter Wert) formatiert ist.
    7. Klicken Stornieren um den Dialog zu schließen.

    Erstellen Sie einen benutzerdefinierten Analytics-Bericht

    Führen Sie die folgenden Schritte aus, um den Analytics-Bericht zu erstellen, der Videos mit Ansichten enthält:

    1. Bewegen Sie sich in Studio in die ANALYTIK Modul.
    2. Klicken Sie im Menü auf der linken Seite auf Benutzerdefiniert.
    3. Verwenden Sie die Konten einschließen Dropdown-Liste, um die gewünschten Konten für den Bericht auszuwählen.
    4. Wählen Sie das Berichtstyp sein One-time.
    5. Wählen Sie a Datumsbereich für den Bericht anhand der Kalender.
    6. Für die Datengruppen wählen Video.
    7. Für die Datenfelder Wählen Sie die drei Felder Video Name , Video Views und Video Impressions::
    8. Unter den Feldern sehen Sie eine Vorschau Ihres Berichts. Beachten Sie, dass die Video-ID automatisch in den Bericht aufgenommen wird:
    9. Lieferung a Berichtsname oder akzeptieren Sie die Standardeinstellung.
    10. Wählen Sie das Format sein CSV.
    11. Wählen Sie, ob Sie eine E-Mail erhalten möchten oder nicht, wenn der Bericht fertig ist. Höchstwahrscheinlich warten Sie nur auf den Bericht, es sei denn, Ihr Konto enthält eine sehr große Anzahl von Videos.
    12. Drücke den Bericht anfordern Taste.
    13. Nach einiger Verzögerung wird der Bericht heruntergeladen.

    Importieren Sie die Datei mit allen Videos

    In diesem Dokument werden Google Text & Tabellen verwendet, da die Funktionalität allen kostenlos zur Verfügung steht. Wenn Sie ein Excel-Benutzer sind, können Sie damit alle in diesem Dokument gezeigten Tabellenkalkulationsaufgaben ausführen.

    In diesem Abschnitt des Dokuments öffnen Sie den aus dem Medienmodul erstellten Bericht über alle exportierten Videos.

    1. Öffnen Sie eine leere Google Text & Tabellen-Tabelle.
    2. Wählen Datei> Importieren> Hochladen Ziehen Sie dann entweder die CSV-Datei für alle Videos (benannt nach Ihrer Konto-ID) an den richtigen Speicherort oder suchen Sie sie aus.
    3. Auf der Datei importieren Dialog, übernehmen Sie die Standardeinstellungen und klicken Sie auf Daten importieren::
    4. Ihre importierten Daten werden ähnlich den folgenden angezeigt:

    Analytics-Bericht importieren

    Sie importieren jetzt den von Ihnen erstellten Analytics-Bericht in ein anderes Blatt (in derselben Tabellengruppe).

    1. Wählen Datei> Importieren> Hochladen Ziehen Sie dann entweder die Analytics-Berichts-CSV-Datei an den richtigen Speicherort oder durchsuchen Sie sie, um sie auszuwählen.
    2. Auf der Datei importieren Dialog, Achten Sie darauf, die zu ändern Speicherort importieren zu Neue Blätter einlegen. Übernehmen Sie die anderen Standardeinstellungen und klicken Sie auf Daten importieren::
    3. Ihre importierten Daten werden ähnlich den folgenden angezeigt:
    4. Beachten Sie, dass am unteren Rand der Tabelle die beiden aufgelisteten Blätter angezeigt werden.

    Erstellen Sie einen benannten Bereich

    Sie fügen jetzt eine Spalte hinzu, die bestimmt, ob ein Video Ansichten hat. Dazu müssen Sie auf die IDs im Analytics-Bericht verweisen (diese Videos mit Ansichten). Um den Vorgang zu vereinfachen, erstellen und benennen Sie jetzt einen Bereich.

    1. Klicken Sie im Analytics-Berichtsblatt auf den Großbuchstaben EIN am oberen Rand der ersten Spalte.
    2. Klicken Sie mit der rechten Maustaste irgendwo in die Spalte und wählen Sie Benannten Bereich definieren ...
    3. Auf der rechten Seite der Tabelle wird ein Fenster geöffnet. Ändern Sie den Standardnamen von NamedRange1 zu IDsWithViews.
    4. Nach dem Drücken der Eingabetaste wird der definierte Bereich angezeigt.

    Spalte hinzufügen, damit keine Ansichten ermittelt werden

    Sie fügen nun eine Spalte hinzu und fügen eine hinzu VLOOKUP() Funktion, die meldet, ob ein bestimmtes Video Ansichten hat.

    1. Kehren Sie zum Blatt mit allen Videos zurück, das mit Ihrer Konto-ID benannt ist.
    2. Klicken Sie auf die Spalte B..
    3. Wählen Sie im Tabellenkalkulationsmenü die Option Einfügen> Spalte links. Dadurch wird eine neue leere Spalte B erstellt.
    4. Klicken Sie in die Zelle B2.
    5. Eingeben =VLOOKUP(. Dies startet die Verwendung der Funktion.
    6. Klicken Sie kurz nach den öffnenden Klammern auf A2 Zelle. Dieser Zellenort wird eingefügt. Dies definiert, wonach die Funktion suchen wird.
    7. Geben Sie nach der Zelle ein Komma ein und geben Sie ein id. Sie werden sehen, dass Sie den benannten Bereich auswählen und ihn auswählen können. Dies definiert, wo die Funktion nach dem Wert in A2 sucht.
    8. Geben Sie nach dem angegebenen Bereich ein Komma ein und geben Sie die Nummer ein 1. Dies definiert den zurückzugebenden Spaltenindex. Für Sie bedeutet dies, dass die ID in der Zelle platziert wird, wenn die ID in A2 im angegebenen Bereich gefunden wird.
    9. Geben Sie nach der Nummer 1 ein Komma ein und geben Sie dann ein FALSE Geben Sie dann die schließenden Klammern ein. Diese Option teilt die VLOOKUP() Funktion Wenn Spalten sortiert sind und der genaue Wert nicht gefunden werden kann, sollte der nächstgelegene Wert zurückgegeben werden. Da Sie dies nicht möchten, FALSE ist der richtige einzugebende Wert.
    10. Stellen Sie sicher, dass Ihre Funktion wie hier gezeigt korrekt definiert ist, und drücken Sie Eingeben. Zelle B2 zeigt entweder die ID von A2 oder an #N/A.
    11. Klicken Sie in Zelle B2 zurück und stellen Sie sicher, dass es markiert ist.
    12. Um die Formel in die gesamte Tabelle zu kopieren, doppelklicken Sie auf das blaue Quadrat unten rechts in der hervorgehobenen Zelle, die derzeit B2 ist. In Spalte B sehen Sie nun entweder IDs aus Spalte A oder #N/A. Wenn Sie die ID sehen, wird Ihnen mitgeteilt, dass das durch diese ID dargestellte Video im Analytics-Bericht gefunden wurde. Mit anderen Worten, es hat Videoansichten. Und wie Sie vermuten können, wenn Sie sehen #N/A Das durch die ID in Spalte A dargestellte Video hat keine Ansichten.

    Löschen Sie Videos ohne Ansichten

    Sie sortieren jetzt die Tabelle so, dass alle Video-IDs ohne Ansichten zusammengefasst werden. Anschließend kopieren Sie diese IDs und formatieren sie zum Löschen von Massenvideos.

    1. Markieren Sie Spalte B und wählen Sie dann im Tabellenkalkulationsmenü die Option aus Daten> Blatt nach Spalte B, Z sortieren -> A. . Dadurch werden alle Videos ohne Ansichten oben in der Tabelle gruppiert, dargestellt durch #N/A in der Zelle.
    2. Sie haben jetzt zwei Möglichkeiten, wie Sie fortfahren sollen. Diese lauten:
      • Wenn Sie eine kleine Anzahl von Videos löschen möchten, können Sie einfach in Studio gehen und manuell in der löschen MEDIEN Modul.
      • Wenn Sie eine erhebliche Anzahl von Videos löschen müssen, möchten Sie möglicherweise die folgenden Schritte ausführen, um das Löschen der Videos zu vereinfachen.
    3. Markieren und kopieren Sie NUR die Video-IDs für die Videos, die Sie löschen möchten. Fügen Sie keine Videos hinzu, denen eine ID oder ein Name fehlt, da dies bedeutet, dass das Video vollständig gelöscht ist.
    4. Öffnen Sie ein Google Doc-Dokument.
    5. Wählen Sie im Dokumentmenü die Option Bearbeiten> Einfügen ohne Formatierung. Sie sehen einfach die Video-IDs in einer Spalte. Um die Videos in großen Mengen zu löschen, müssen sie im Format vorliegen ID: 12345678 ID: 23456789 .... Fahren Sie mit diesen Schritten fort, anstatt die Liste manuell zu bearbeiten.
    6. Wählen Bearbeiten> Suchen und ersetzen ...
    7. Für die Finden Wert eingeben \n. Dies stellt einen Wagenrücklauf dar.
    8. Für die Ersetzen mit Wert Geben Sie ein Leerzeichen gefolgt von ein id:. Der Raum vor dem id: ist entscheidend für die korrekte Formatierung.
    9. Überprüf den Übereinstimmung mit regulären Ausdrücken Kontrollkästchen.
    10. Drücke den Alles ersetzen Klicken Sie auf die Schaltfläche, und schließen Sie den Dialog.
    11. Stellen Sie sicher, dass Ihre Liste der Video-IDs jetzt ähnlich wie folgt formatiert ist: 
        1754261702001 id:4082515861001 id:5185914204001 id:5324448023001
  id:1754261492001 id:2180442442001 id:2535301903001 id:4713495235001
  id:4874718207001 id:4997278349001 id:5686632029001 id:5998203290001
  id:5998205402001 id:5998658922001 id:5998662608001 id:
    12. Um die Formatierung abzuschließen, müssen Sie eine platzieren id: vor der ersten Video-ID. Dann entfernen Sie die id: am Ende der Liste, die keine übereinstimmende tatsächliche Video-ID hat.
    13. Kopieren Sie die korrekt formatierte Liste der Video-IDs. 
        id:1754261702001 id:4082515861001 id:5185914204001 id:5324448023001
  id:1754261492001 id:2180442442001 id:2535301903001 id:4713495235001
  id:4874718207001 id:4997278349001 id:5686632029001 id:5998203290001
  id:5998205402001 id:5998658922001 id:5998662608001
    14. Öffnen Sie Studios MEDIEN Modul.
    15. Fügen Sie Ihre Liste der Video-IDs in das Suchfeld ein:
      Wenn Sie eine sehr große Anzahl von Videos entfernen müssen, ist es am besten, mehrere Sätze von IDs anstelle aller IDs gleichzeitig zu verwenden. Beim Testen haben 100 Videos gleichzeitig erfolgreich funktioniert.
    16. Drücken Sie Eingeben.
    17. Sie sehen nun alle aufgelisteten Videos, die mit den zuvor ausgewählten IDs übereinstimmen. Sie können jetzt einzeln zu löschende Videos auswählen oder das Kontrollkästchen Alle auswählen aktivieren und die Videos löschen.
