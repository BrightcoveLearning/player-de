Brightcove Spielerinformationen Modal

{{ page.description }}

Spielerinformationen anzeigen

Sie zeigen die Spielerinformationen an, indem Sie mit der rechten Maustaste auf den Spieler klicken und Spielerinformationen auswählen, wie in diesem Screenshot gezeigt.

Hinweis: Auf Mobilgeräten können Sie mit dem langen Drücken das Modal öffnen.

Sobald Sie Spielerinformationen ausgewählt haben, wird das modale Fenster mit Informationen über den Player angezeigt, wie in diesem Screenshot gezeigt.

Hinweis: Im Abschnitt Plugins werden nur von Brightcove gelieferte Plugins angezeigt, die Teil der Konfiguration sind. Wenn Sie ein Brightcove-Plugin im JavaScript einer HTML-Seite hinzufügen, wird es NICHT in der Liste angezeigt. Von Kunden entwickelte Plugins, auch wenn sie als Teil der Konfiguration des Players aufgeführt sind, werden NICHT angezeigt.

Schließen Sie das modale Fenster

Sie schließen das Modal entweder durch Drücken der ESC-Taste oder klicken auf das Schließen -Symbol oben links.

Ändern Sie das Kontextmenü

Sie können das Menü ändern, das mit der rechten Maustaste angezeigt wird, das Kontextmenü genannt wird. Um dies zu tun, musst du die Konfiguration deines Players ändern. Sie können dies auf zwei Arten tun:

  • Verwenden des JSON-Editors von Studio aus dem PLAYERS-Modul.
  • Ein Tool zum Durchführen der Konfigurationsänderung ist curl.

Die beiden Änderungen, die Sie vornehmen können und wie Sie sie beeinflussen können, werden hier angezeigt.

Hinweis: Die folgenden curl-Anweisungen verwenden die Player-Management-API, um die Spielerkonfiguration zu ändern. In den curl-Anweisungen sehen Sie Werte von $EMAIL, $ACCOUNT_ID und $PLAYER_ID. Sie müssen entweder Umgebungsvariablen mit diesen Namen erstellen und entsprechende Werte zuweisen oder die Literalwerte für die Variablen angeben.

Verhindern, dass das Menü erscheint

Wenn Sie verhindern möchten, dass das Kontextmenü der Spielerinformationen überhaupt angezeigt wird, können Sie einen der folgenden Schritte ausführen:

Verwenden Sie den JSON-Editor

Ändern Sie die Konfiguration des Spielers, indem Sie die hervorgehobene Eigenschaft hinzufügen (speichern Sie die Änderung unbedingt):

Verwenden Sie curl

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
      "context_menu": false
  }' \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration
Hinweis: Immer wenn Sie eine Änderung an einem Spieler mit curl vornehmen, müssen Sie den Player erneut veröffentlichen. Sie können dies in Studio tun, und Sie werden sehen, dass Ihr Player in der als zur Veröffentlichung erforderlich markiert ist Status Säule:

Du kannst curl auch verwenden, um deinen Player zu veröffentlichen:

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request POST \
    https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/publish

Verhindern Sie die Menüoption Brightcove

Wenn Sie verhindern möchten, dass die Menüoption Über Brightcove angezeigt wird, können Sie einen der folgenden Schritte ausführen:

Verwenden Sie den JSON-Editor

Ändern Sie die Konfiguration des Spielers, indem Sie die hervorgehobene Eigenschaft hinzufügen (speichern Sie die Änderung unbedingt):

Verwenden Sie curl

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request PATCH \
  --data '{
    "context_menu": {
      "brightcove_branding": false
    }
  }' \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/configuration
Hinweis: Immer wenn Sie eine Änderung an einem Spieler mit curl vornehmen, müssen Sie den Player erneut veröffentlichen. Du kannst das in Studio tun und du wirst sehen, dass dein Spieler in der Spalte Status als veröffentlicht werden muss:

Du kannst curl auch verwenden, um deinen Player zu veröffentlichen:

curl \
  --header "Content-Type: application/json" \
  --user $EMAIL \
  --request POST \
  https://players.api.brightcove.com/v2/accounts/$ACCOUNT_ID/players/$PLAYER_ID/publish
