Erstellen Sie ein Netzwerkprotokoll aus Chrome

Die folgenden Schritte führen Sie durch die Verwendung von Chrome zum Erstellen eines Netzwerkprotokolls.

  1. Öffnen Sie ein Chrome-Browserfenster oder eine Registerkarte.
  2. Um sicherzustellen, dass zwischengespeicherte Informationen im Browser das Netzwerkprotokoll nicht beeinflussen, löschen Sie zunächst die Browserdaten. Um diesen Prozess zu beginnen, von der Optionen Menü (drei vertikale Punkte) Klicken Sie auf Browserdaten löschen... Möglichkeit.
  3. Treffen Sie nun in der eigentlichen Benutzeroberfläche die Auswahl wie im folgenden Screenshot gezeigt und klicken Sie auf Browserdaten löschen Taste.
  4. Im Browser wieder aus dem Optionen Menü auswählen Entwicklerwerkzeuge.
  5. Wähle aus Netzwerk Registerkarte in den Entwicklertools.
  6. Fügen Sie die URL auf die Seite mit dem Brightcove Player, den Sie beheben, in die Adressleiste des Browsers ein, und drücken Sie dann die Eingabetaste, um die Seite zu laden. Auf der Registerkarte Netzwerk wird sofort der Netzwerkverkehr angezeigt.
  7. Wenn Netzwerkverkehr benötigt wird, wenn der Player ein Video abspielt, starten Sie das Video.
  8. Wenn Sie den erforderlichen Netzwerkverkehr haben, klicken Sie mit der rechten Maustaste in den Verkehrsbereich und wählen Sie Als HAR mit Inhalt speichern Möglichkeit.
  9. Speichern Sie die Datei und senden Sie sie an das entsprechende Support-Personal.

Erstellen Sie ein Netzwerkprotokoll aus Firefox

Die folgenden Schritte führen Sie durch die Verwendung von Firefox zum Erstellen eines Netzwerkprotokolls.

  1. Öffnen Sie ein Firefox-Browserfenster oder eine Registerkarte.
  2. Um sicherzustellen, dass zwischengespeicherte Informationen im Browser das Netzwerkprotokoll nicht beeinflussen, löschen Sie zunächst die Browserdaten. Um diesen Prozess zu beginnen, von der Optionen Menü (drei vertikale Linien), wählen Sie die Einstellungen Option und dann Privatsphäre & Sicherheit.
  3. Runterscrollen. Rechts von Zwischengespeicherter Webinhalt , wähle aus Frei Jetzt Taste.
  4. Schließe Einstellungen Tab.
  5. Im Browser wieder aus dem Optionen Menü auswählen Web-Entwickler und dann auswählen Netzwerk.
  6. Fügen Sie die URL auf die Seite mit dem Brightcove Player, den Sie beheben, in die Adressleiste des Browsers ein, und drücken Sie dann die Eingabetaste, um die Seite zu laden. Auf der Registerkarte Netzwerk wird sofort der Netzwerkverkehr angezeigt.
  7. Wenn Netzwerkverkehr benötigt wird, wenn der Player ein Video abspielt, starten Sie das Video.
  8. Wenn Sie den erforderlichen Netzwerkverkehr haben, klicken Sie mit der rechten Maustaste in den Verkehrsbereich und wählen Sie die aus Alles als HAR speichern Möglichkeit.
  9. Speichern Sie die Datei und senden Sie sie an das entsprechende Support-Personal.

Erstellen Sie ein Webarchiv aus Safari

Die folgenden Schritte führen Sie durch die Verwendung von Safari zum Erstellen einer Webarchivdatei (.webarchive). Dies ist nicht dasselbe wie eine HAR-Datei.

In Safari gibt es keine Möglichkeit, eine HAR-Datei zu generieren. Es wird daher empfohlen, entweder Chrome oder Firefox zu verwenden, um Ihre Netzwerkprotokolldatei zu generieren.
  1. Öffnen Sie ein Safari-Browserfenster oder eine Registerkarte.
  2. Um sicherzustellen, dass zwischengespeicherte Informationen im Browser das Netzwerkprotokoll nicht beeinflussen, löschen Sie zunächst die Browserdaten. Wenn Sie das nicht sehen Entwickeln Menü im oberen Menü, dann wählen Sie im Safari-Menü die Option Safari und dann Einstellungen.
  3. Wählen Sie im Dialogfeld "Einstellungen" die Option "Auswählen" Fortgeschrittene und dann auswählen Menü "Entwickeln" in der Menüleiste anzeigen.
  4. Wählen Sie in der Safari-Menüleiste die Option Entwickeln und dann auswählen Leere Caches.
  5. Fügen Sie die URL auf die Seite mit dem Brightcove Player, den Sie beheben, in die Adressleiste des Browsers ein, und drücken Sie dann die Eingabetaste, um die Seite zu laden.
  6. Wählen Sie in der Safari-Menüleiste die Option Entwickeln und dann auswählen Web Inspector anzeigen.
  7. Wählen Sie im Webinspektor die Option aus Netzwerk Tab.
  8. Laden Sie Ihre Webseite im Browser neu. Auf der Registerkarte Netzwerk wird sofort der Netzwerkverkehr angezeigt.
  9. Wenn Netzwerkverkehr benötigt wird, wenn der Player ein Video abspielt, starten Sie das Video.
  10. Wenn Sie den erforderlichen Netzwerkverkehr haben, wählen Sie die Laden Sie das Webarchiv herunter Symbol zum Generieren einer HAR-Datei.
  11. Speichern Sie die Datei. Diese Datei enthält möglicherweise nicht die Netzwerkinformationen, die Support-Mitarbeiter möglicherweise benötigen, um bei Ihrem Problem zu helfen.
