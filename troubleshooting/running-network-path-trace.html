Ausführen einer Netzwerkpfadverfolgung

{{ page.description }}

Überblick

Das traceroute Das Diagnosetool verfolgt den Pfad von Ihrem Computer zur betreffenden Website. Das Tool sendet eine Folge von Paketen über das Netzwerk und empfängt Antworten von jedem Router im Pfad. Auf diese Weise wird festgestellt, ob auf dem Pfad Probleme auftreten.

Wenn Sie mit dem Support arbeiten, werden Sie möglicherweise aufgefordert, die Informationen anzugeben, die vom Verfolgen des Netzwerkpfads zur betreffenden Website stammen.

Rückverfolgung von macOS oder Linux

Gehen Sie folgendermaßen vor, um eine Netzwerkpfadverfolgung auszuführen:

  1. Öffnen Sie ein Fenster für die Terminal-App.
  2. Geben Sie bei der betreffenden Website den folgenden Befehl ein. 
    traceroute your_domain.com

    Wenn Sie wissen, dass Ihr Netzwerk Internet Protocol Version 6 (IPv6) verwendet, geben Sie den folgenden Befehl ein:

    traceroute6 your_domain.com
  3. Sie sollten den Verkehrspfad sehen, der zu Ihrer Website geführt wird. Es wird aufgebaut, wenn Antworten von den Routern auf dem Weg empfangen werden. Die erste Zeile repräsentiert Ihren Router. Die nächste Zeile stammt von Ihrem ISP und wird fortgesetzt, bis die Pakete Ihre Website erreichen. Ihre Ausgabe sollte folgendermaßen aussehen:
  4. Machen Sie sich keine Sorgen darüber, die Ausgabe zu verstehen. Wählen Sie in Ihrem Terminalfenster aus Bearbeiten > Wählen Sie Alle. Kopieren Sie dann die Ausgabe in Ihre Zwischenablage.
  5. Fügen Sie die Ausgabe in eine E-Mail ein und senden Sie sie an das entsprechende Support-Personal.

Rückverfolgung von Windows

Gehen Sie folgendermaßen vor, um eine Netzwerkpfadverfolgung auszuführen:

  1. Öffne das Start Menü und wählen Lauf.
  2. Art cmd und wählen Sie OK.
  3. Dadurch wird die Eingabeaufforderung geöffnet. Geben Sie bei der betreffenden Website den folgenden Befehl ein. 
    tracert your_domain.com

    Wenn Sie wissen, dass Ihr Netzwerk Internet Protocol Version 6 (IPv6) verwendet, geben Sie den folgenden Befehl ein:

    tracert -6 your_domain.com
  4. Sie sollten den Verkehrspfad sehen, der zu Ihrer Website geführt wird. Es wird aufgebaut, wenn Antworten von den Routern auf dem Weg empfangen werden. Die erste Zeile repräsentiert Ihren Router. Die nächste Zeile stammt von Ihrem ISP und wird fortgesetzt, bis die Pakete Ihre Website erreichen.
  5. Machen Sie sich keine Sorgen darüber, die Ausgabe zu verstehen. Kopieren Sie die Ausgabe in Ihre Zwischenablage.
  6. Fügen Sie die Ausgabe in eine E-Mail ein und senden Sie sie an das entsprechende Support-Personal.
