Anzeigen der Player-Konfiguration

{{ page.description }}

Konfiguration anzeigen

Befolgen Sie diese Schritte, um die Konfiguration eines Spielers anzuzeigen:

  1. Gehen Sie im Studio zu SPIELER Modul.
  2. Klicken Sie auf das Video, dessen Konfiguration Sie anzeigen möchten.
  3. Drücke den Veröffentlichen & Einbetten Taste.
  4. Stellen Sie sicher, dass alle Änderungen veröffentlicht werden.
  5. Klick auf das Spieler-URL (wie unten gezeigt). Sie sehen den Player in einem Browser geöffnet.
  6. Überprüfen Sie die URL im Browser. Es sieht ungefähr so aus: 
    https://players.brightcove.net/1752604059001/SAJo2wjVo_default/index.html
  7. Ändere das index.html in der URL zu config.json.
  8. Die geänderte URL zeigt die Konfiguration des Players an:
    Hinweis: Der resultierende JSON wird möglicherweise nicht so gut angezeigt wie im Screenshot. Sie können eine JSON-Formatierungserweiterung/ein JSON-Add-On für Ihren Browser finden, um die JSON-Formatierung zu unterstützen.

Konfiguration ändern

Obwohl das Ändern der Konfiguration den Rahmen dieses Dokuments sprengt, sind einige Kommentare zu diesem Thema angebracht.

  • Studio bietet eine Benutzeroberfläche, mit der Sie viele Aspekte der Konfiguration des Players im Internet ändern können SPIELER Modul.
  • Für die vollständige Kontrolle über die Konfiguration des Players Player Management API kann verwendet werden. Beachten Sie, dass dies für Entwickler gedacht ist und diese verwendet locken Aussagen ausführlich. Die Schritt für Schritt: Spielerverwaltung Dokument bietet eine Einführung in den Prozess.
