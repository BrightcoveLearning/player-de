Spielereigenschaften

Die Brightcove Player-Eigenschaften enthalten Informationen zu Ihrem bestimmten Player. Einzelheiten zum Einstellen des Player-Skins und der Version finden Sie in der Skins und Player-Versionen Dokument.

VERSION

Das bc.VERSION Die Eigenschaft gibt die Version des Players an, wie sie in der Player-Konfiguration definiert ist.

Hier ist ein Beispiel:

    <video-js id="video_1" style="width: 640px; height: 360px;"
    data-video-id="4029697544001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js"
    controls></video-js>
    <script src="https://players.brightcove.net/1752604059001/default_default/index.min.js"></script>


    <!-- custom script -->
    <script type="text/JavaScript">
        var myPlayer;

	videojs.getPlayer('video_1').ready(function() {
            myPlayer = this;
            console.log(bc.VERSION);
            console.log(bc.SKIN);
        });
    </script>
Zurückgegebener Wert:

Der in diesem Fall zurückgegebene Wert ist 5.1.7

Dieser Wert stammt aus der Player-Konfiguration, die durch eine Vorschau des Players im Browser und Wechsel index.html zu gefunden werden kann config.json.

HAUT

Das bc.SKIN Die Eigenschaft gibt die Skin an, die auf den Player angewendet wird.

Hier ist ein Beispiel:

    <video-js id="video_1" style="width: 640px; height: 360px;"
    data-video-id="4029697544001"
    data-account="1752604059001"
    data-player="default"
    data-embed="default"
    class="video-js"
    controls></video-js>
    <script src="//players.brightcove.net/1752604059001/default_default/index.min.js"></script>

    <!-- custom script -->
    <script type="text/JavaScript">
        var myPlayer;

	  videojs.getPlayer('video_1').ready(function() {
            myPlayer = this;
            console.log(bc.VERSION);
            console.log(bc.SKIN);
        });
    </script>
Zurückgegebener Wert:

Der in diesem Fall zurückgegebene Wert ist Null Dies bedeutet, dass die Luna-Haut auf diesen Spieler aufgetragen wird. Hier ist eine Zusammenfassung der Rückgabewerte aus dem bc.SKIN Eigentum:

bc.SKIN-Wert Haut aufgetragen
falsch Keine Haut, so dass Sie mit Ihren eigenen Stilen überschreiben können
Graphit Graphithaut
alle anderen Werte (oder deren Fehlen), einschließlich undefiniert, null, wahr usw. Luna-Haut (Standard-Haut)

Dieser Wert stammt aus der Player-Konfiguration, die durch eine Vorschau des Players im Browser und Wechsel index.html zu gefunden werden kann config.json.

Hinweis: Das bc.SKIN Eigenschaft wird nur ausgefüllt, wenn die "skin" Die Eigenschaft in der Player-Konfiguration hat einen expliziten Wert. Andernfalls ist es undefiniert.
