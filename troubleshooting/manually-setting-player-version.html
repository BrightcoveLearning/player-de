Manuelles Einstellen der Player-Version

{{ page.description }}

Warum die Version manuell ändern?

Möglicherweise müssen Sie auf eine frühere Player-Version zurücksetzen, da ein Player-Upgrade ein Problem verursacht. Sie können auf eine frühere Version zurücksetzen, bis der Test abgeschlossen ist, um den Grund für das Problem zu finden und es behoben ist.

Eine ausführliche Erläuterung der Player-Versionseinstellungen in Studio finden Sie im Einstellen des Player-Update-Modus und -Version dokumentieren.

Prozess

Mit diesen Schritten können Sie eine Player-Version manuell festlegen.

  1. Zunächst möchten Sie den Player so einstellen, dass er manuell aktualisiert wird. Dadurch wird verhindert, dass eine Änderung der Player-Version durch ein Player-Update überschrieben wird. Gehen Sie dazu zu Studio, SPIELER Modul und in der Spielerinformationen Abschnitt setzen die Aktualisierungsmodus zu Handbuch. Speichern Sie natürlich Ihre Änderungen.
  2. Jetzt stellen Sie die Player-Version manuell mit ein JSON-Editor in dem SPIELER Modul. Dies ist ein Beispiel für eine Player-Konfiguration, bevor eine Versionsänderung vorgenommen wird:
    Beachten Sie das "locked": true Attribut ist ein Ergebnis von Schritt 1.
    In dem JSON-Editor , ändern Sie die Ausführung Attribut Ihrer gewünschten Version. Speichern Sie Ihre Änderungen und veröffentlichen Sie Ihren Player.

Vergessen Sie nicht, sobald das Problem behoben ist, Ihren Player wieder so zu ändern, dass er automatisch aktualisiert wird.
