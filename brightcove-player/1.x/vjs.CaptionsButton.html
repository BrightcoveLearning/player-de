optionen (obj)

Tiefes Zusammenführen von Options-Objekten

Immer wenn eine Eigenschaft ein Objekt für beide Optionsobjekte ist, werden die beiden Eigenschaften mit vjs.obj.DeepMerge zusammengeführt.

Dies wird zum Zusammenführen von Optionen für untergeordnete Komponenten verwendet. Wir möchten, dass es einfach ist, einzelne Optionen für eine untergeordnete Komponente zu überschreiben, ohne alle anderen Standardoptionen neu schreiben zu müssen.

parent.prototype.options_ = { children: { 'childOne': { 'foo': 'bar', 'asdf': 'fdsa' }, 'childTwo': {}, 'ChildThree': {}}} newOptions = { children: { 'childOne': { 'foo': 'baz', 'abc': '123' } 'ChildTwo': null, 'ChildFour': {} }} this.options (newOptions);

ERGEBNIS

{ children: { 'childOne': { 'foo': 'baz', 'asdf': 'fdsa', 'abc': '123' }, 'childTwo': null, // Disabled. Wird nicht initialisiert. 'ChildThree': {}, 'ChildFour': {}}}
PARAMETER:
  • obj Objektobjekt neuer Optionswerte
GIBT ZURÜCK:
  • Objekt Ein NEUES Objekt von this.options_ und obj fusioniert
