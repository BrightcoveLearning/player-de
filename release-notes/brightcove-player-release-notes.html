Hinweis: Standardmäßig werden Updates automatisch auf alle Spieler angewendet. Um eine automatische Aktualisierung eines Players zu verhindern, müssen Sie ihn auf manuelle Aktualisierung einstellen (auch als Sperren eines Players bezeichnet). Sie können dies im Studio Player-Modul (externe Website) oder mit der Player Management-API tun.

Das Abmelden von automatischen Updates sollte eine vorübergehende Maßnahme sein, die angewendet wird, während Sie alle Probleme beheben, die vom Update erstellt werden. Wenn Sie die Spieler nicht auf die aktuelle Version aktualisieren, verpassen Sie neue Funktionen und Bugfixes, und ältere Versionen des Players werden irgendwann nicht mehr unterstützt

Weitere Versionshinweise für ältere Spieler finden Sie im Archiv.

Weitere Versionshinweise zu älteren Plugins finden Sie auf den einzelnen Plugin-Seiten.
