Verwenden von Player-Verbesserungen für Live-SSAI mit Brightcove Player

{{ page.description }}

Einleitung

Mit den Player-Erweiterungen für Live-SSAI von Brightcove können Sie die Werbeunterbrechungen in Live-Streams mit Anzeigenzählungen, Anzeigen-Countdown-Timern, Click-through-Anzeigen und Companion-Anzeigen verbessern. Diese Live-Streams können mit Brightcove Player abgespielt werden.

Wenn Sie diese Funktion noch nicht kennen, lesen Sie Folgendes:

  • Überblick: Spielerverbesserungen für Live-SSAI
  • Implementieren von Spielerverbesserungen für Live-SSAI

Schritte

Gehen Sie folgendermaßen vor, um Live-Streams mit Player-Verbesserungen mit dem Brightcove Player abzuspielen:

  1. Erstellen oder wählen Sie einen vorhandenen Player aus den Video Cloud Studios Spieler Modul.

  2. Brightcove Player muss mit dem SSAI-Plugin konfiguriert werden. Folge dem Fügen Sie das SSAI-Plugin hinzu Schritte beschrieben in der SSAI programmgesteuert implementieren Abschnitt des SSAI-Plugin-Dokuments.

    Für Player-Erweiterungen mit Live-SSAI benötigt Brightcove Player das SSAI-Plugin (videojs-ssai). Derzeit können Sie Video Cloud Studio nicht zum Hinzufügen des SSAI-Plug-ins zu Ihrem Player verwenden, da die Anzeigenkonfigurations-ID erforderlich ist.
  3. Folgen Sie dem Workflow in der Implementieren von Player-Verbesserungen für Live-SSAI Dokument, bis einschließlich der Erstellung Ihres Wiedergabetokens.
  4. Wählen Sie eine Implementierung:
    • Standard-Einbettungscode
    • Erweiterter Einbettungscode
    • Live-SSAI-Quelle
  5. Fahren Sie mit dem Workflow in der Implementieren von Spielerverbesserungen für Live-SSAI dokumentieren.

Standard-Einbettungscode

Beim Veröffentlichen eines Videos aus Video Cloud Studios Medien Modul können Sie den Standard-Einbettungscode verwenden, um Ihren Live-Stream abzuspielen. Folgen Sie diesen Schritten:

  1. Wählen Sie im Medienmodul Ihr Remote-Video für das Live-Streaming aus und veröffentlichen Sie es mit einem SSAI-fähigen Player.
  2. Kopieren Sie den Standardeinbettungscode und fügen Sie ihn in Ihre Webseite ein.

  3. In dem <iframe> Tag, sollten Sie eine sehen src Attribut. Fügen Sie am Ende das hinzu adConfigId URL-Parameter mit dem Wert Ihres Wiedergabetoken.

    <iframe src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/
index.html?videoId=6089818309001&adConfigId=live.aaB61234aabbccdd567890"
  allowfullscreen=""
  allow="encrypted-media"
  width="960" height="540">
</iframe>
  4. Dein Spieler ist bereit, deinen Livestream abzuspielen.

Erweiterter Einbettungscode

Beim Veröffentlichen eines Videos aus Video Cloud Studios Medien Modul können Sie den erweiterten Einbettungscode verwenden, um Ihren Live-Stream abzuspielen. Folgen Sie diesen Schritten:

  1. Wählen Sie im Medienmodul Ihr Remote-Video für das Live-Streaming aus und veröffentlichen Sie es mit einem SSAI-fähigen Player.
  2. Kopieren Sie den erweiterten Einbettungscode und fügen Sie ihn in Ihre Webseite ein.

  3. In dem <video-js> Tag, fügen Sie ein data-ad-config-id Attribut mit dem Wert Ihres Wiedergabetoken.

    <video-js
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id="6089818309001"
  data-ad-config-id="live.aaB61234aabbccdd567890"
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
  4. Dein Spieler ist bereit, deinen Livestream abzuspielen.

Live-SSAI-Quelle

Im Gegensatz zu VOD-SSAI-Quellen enthalten Live-SSAI-Quellen keinen anfänglichen VMAP. Dies liegt daran, dass Live-SSAI-Streams nicht im Voraus wissen, wann Werbeunterbrechungen enthalten sein werden. Stattdessen werden Werbeunterbrechungen von Cue-Punkten ausgelöst, die Sie dem Live-Job hinzufügen.

Brightcove Live SSAI-Streams beginnen mit einer Standard-HLS-Manifest-URL mit einem bestimmten Namensschema:

https://.../playlist_ssaiM.m3u8

Das SSAI-Plugin sucht nach dem _ssaiM.m3u8 Suffix und der normale HLS-MIME-Typ

Beim Veröffentlichen eines Videos aus Video Cloud Studios Medien Modul können Sie einen Live-SSAI-Stream wie folgt abspielen:

  1. Wählen Sie im Medienmodul Ihr Remote-Video für das Live-Streaming aus und veröffentlichen Sie es mit einem SSAI-fähigen Player.
  2. Kopieren Sie den erweiterten Einbettungscode und fügen Sie ihn in Ihre Webseite ein.

  3. Fügen Sie Ihrem Seitencode den folgenden Code hinzu:

    <video-js id="myPlayerID"
  data-account="6027103979001"
  data-player="IqljF4Olr"
  data-embed="default"
  controls=""
  data-video-id=""
  data-playlist-id=""
  data-application-id=""
  width="960" height="540">
</video-js>
<script src="https://players.brightcove.net/6027103979001/IqljF4Olr_default/index.min.js"></script>
<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.src({
      type: 'application/x-mpegURL',
      src: 'https://path/to/playlist_ssaiM.m3u8'
    })
  });
</script>
  4. Dein Spieler ist bereit, deinen Livestream abzuspielen.
