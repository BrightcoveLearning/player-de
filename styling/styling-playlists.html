Styling von Wiedergabelisten

In diesem Thema lernen Sie, wie Sie Wiedergabelisten formatieren.

Überblick

Wenn Sie das Aussehen von Wiedergabelisten ändern möchten, können Sie dies mithilfe von CSS tun. In diesem Dokument sehen Sie eine Reihe von Stilen, die Sie ändern können. Natürlich gibt es zahlreiche mögliche Änderungen, und Sie müssen den richtigen Selektor finden, um Ihre Aufgabe zu erfüllen.

Die folgenden Dokumente sind wichtige Informationen, um das Bild der Verwendung von Playlists zu vervollständigen:
  • Wiedergabelisten implementieren: Dieses Dokument zeigt, welche Entscheidungen Sie über die Verwendung von Wiedergabelisten treffen müssen und wie Sie den Spielercode erhalten, den Sie benötigen, um sie tatsächlich auf Ihrer HTML-Seite zu erhalten.
  • Playlist UI-Plugin: In diesem Dokument werden die verfügbaren Optionen beschrieben, um das Erscheinungsbild der Wiedergabelistenfunktionalität zu verbessern.

Styling einer vertikalen Wiedergabeliste

Das folgende Diagramm fasst die für eine vertikale Wiedergabeliste geänderten Stile zusammen. Deinem Diagramm folgt eine Tabelle, in der die Stile aufgeführt sind.

Steuere die Farbe des Textes auf der Wiedergabeliste .vjs-mouse.vjs-playlist {
  farbe: #E9D2B3;
}
Kontrolliere die Farbe des
Leerzeichen zwischen Playlist-Elementen		 .vjs-mouse.vjs-playlist {
  Hintergrundfarbe: #161D6E;
}
Steuern Sie die Farbe des Dauertextes .vjs-playlist .vjs-playlist-duration {
  farbe: #9BB4DA;
}
Steuern Sie die Farbe des Dauerhintergrunds .vjs-Wiedergabeliste .vjs-Wiedergabeliste-Dauer {
  Hintergrundfarbe: #1160AA;
}
Steuert den Abstand zwischen Playlist-Elementen .vjs-playlist-vertikal .vjs-playlist-item {
  Rand-oben: 20px;
}
Um einen Rand über dem ersten Playlist-Element zu vermeiden, sollten Sie den Randeinstellungsstil wie folgt befolgen: 
.vjs-playlist-vertical .vjs-playlist-item:first-child {
  margin-top: 0px;
}

  farbe: #E9D2B3;
}
Kontrolliere die Farbe des
Leerzeichen zwischen Playlist-Elementen		 .vjs-mouse.vjs-playlist {

}
Steuern Sie die Farbe des Dauertextes .vjs-Wiedergabeliste .vjs-Wiedergabeliste-Dauer {

}
Steuern Sie die Farbe des Dauerhintergrunds .vjs-Wiedergabeliste .vjs-Wiedergabeliste-Dauer {
  Hintergrundfarbe: #1160AA;

Steuert den Abstand zwischen Playlist-Elementen .vjs-playlist-horizontal .vjs-playlist-item {

Um einen Rand links vom ersten Playlist-Element zu vermeiden, sollten Sie den Randeinstellungsstil wie folgt befolgen: 
.vjs-playlist-horizontal .vjs-playlist-item:first-child {
  margin-left: 0px;
}

Bildlaufleiste ausblenden

Eine Bildlaufleiste in der horizontalen Wiedergabeliste wird nicht benötigt, wird jedoch angezeigt. Wenn Sie die Bildlaufleiste ausblenden möchten, führt der folgende Stil genau das aus:

div.vjs-playlist.vjs-playlist-horizontal {
  overflow-y: hidden;
}
