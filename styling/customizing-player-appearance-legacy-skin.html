Anpassen des Spielerauftritts - Legacy Skin

Dieses Dokument enthält Informationen zur Brightcove Player 1.x-Skin (Graphit), nicht zur aktuellsten Skin. Dieser Übergang zur neueren Haut fand in der zweiten Dezemberwoche 2015 statt. Die aktuellste Version des Dokuments finden Sie in diesem Dokument.

Dieses Thema ist eine Einführung in das Anpassen des Erscheinungsbilds des neuen Players / styling / customizing-player-Erscheinungsbild.html. Der Inhalt verwendet die Entwicklungstools von Chrome, aber alle modernen Browser verfügen über eine gleichwertige Funktionalität. Eine kurze Einführung in die Verwendung von Entwicklungstools in verschiedenen Browsern finden Sie in der Grundlagen des JavaScript-Debuggens Dokument.

Eigenes Skin erstellen

Wenn Sie den Standard-Skin nicht verwenden und Ihren eigenen erstellen möchten, ist dies mit der Player Management API möglich. Der Player-Konfigurationshandbuch enthält die notwendigen Informationen.

Spieler

Hinweis: Dieses Dokument setzt einige Kenntnisse über CSS und die Verwendung der Entwicklungstools eines Browsers voraus, um die Klassen von HTML-Elementen anzuzeigen und das anwendbare CSS zu untersuchen. Wenn Sie nicht über die erforderlichen Grundkenntnisse verfügen, ist es am besten, wenn Sie die durcharbeiten Schnellstart: Spieleranpassung – Legacy-Skin.

Sie können das Aussehen des Spielers ändern, müssen aber zuerst eine Möglichkeit haben, es anzugehen. Um dies zu sehen, durchsuchen Sie eine HTML-Seite mit dem eingefügten Player-Code_in_in_page. In dem Elemente Suchen Sie im Abschnitt der Entwicklungstools den ersten <div> und Sie werden sehen, dass es eine Klasse mit dem Wert hat video-js unter anderem zugewiesen.

Sobald Sie dies wissen, können Sie Stile verwenden, um den Spieler selbst zu ändern. Um beispielsweise die Größe des Players zu ändern, können Sie diesen Stil verwenden:

<style>
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

iframe-Spieler

Wenn Sie die Iframe-Implementierung des Players verwenden, ist die Situation anders. Sie sehen immer noch den Spieler mit dem video-js Klasse, aber es wird natürlich innerhalb des Iframes sein.

Die von Ihnen erstellten Stile funktionieren weiterhin mit dem Player im Iframe, aber eine CSS-Datei muss erstellt und mit dem Player verknüpft werden, der Studio verwendet. Sie tun dies, indem Sie zum gehen Spieler Klicken Sie dann auf das Video, mit dem Sie das CSS verknüpfen möchten, und klicken Sie dann auf Plugins Abschnitt fügen Sie den Pfad Ihrer CSS-Datei hinzu.

Wenn Sie den Iframe selbst anpassen möchten, können Sie dies tun, indem Sie a hinzufügen class Attribut auf die iframe Element (Zeile 27). In diesem Fall ist der Klassenwert videoFrame. Verwenden Sie dann ein <style> Tag, um nach Belieben zu ändern. Im Beispiel unten die width und height geändert werden.

<!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Customize iframe</title>

  <!-- Page styles -->
  <style>
    .videoFrame {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <iframe class="videoFrame" src='//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.html' allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
</body>

</html>

Schaltfläche „Spielen“

Wenn Sie das Erscheinungsbild des Play-Buttons ändern möchten, müssen Sie zuerst wissen, wie Sie ihn ansprechen können. Stellen Sie in Chrome sicher, dass die Entwicklungswerkzeuge geöffnet sind und dann mit der rechten Maustaste auf die Schaltfläche klicken. Wählen Sie aus der angezeigten Auswahl Element prüfen.

In dem Elemente Im Abschnitt der Entwicklungstools sehen Sie den HTML-Code, der dem Schaltflächenelement entspricht.

Aus dem oben hervorgehobenen Code sehen Sie die Klasse, die der Schaltfläche zugewiesen ist vjs-big-play-button. Die Farbe des Play-Buttons (passend zum Tiger) und die Deckkraft werden mit diesen Stilen geändert:

  <style>
    .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Man würde denken, dass dieser Ansatz funktionieren würde, aber das tut er nicht. Wenn Sie mit CSS arbeiten, müssen Sie sich der Eigenschaftsspezifität bewusst sein. Dies bezieht sich darauf, wie der Browser anhand ihrer Relevanz bestimmt, welche Eigenschaftswerte auf ein Element angewendet werden.

Eine Möglichkeit, die Schaltflächenänderungen anzuwenden, besteht darin, auf eine ID in der CSS-Auswahl zu verweisen. Dies liegt daran, dass IDs eine hohe CSS-Spezifität aufweisen. Weitere Einzelheiten finden Sie in der CSS-Spezifität Sektion

Fügen Sie zunächst eine hinzu id Attribut auf die div Element für den Spielercode.

    <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

    <script src="//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.min.js"></script>

Referenzieren Sie dann den großen Play-Button des Spielers im CSS wie folgt:

  <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Hier ist das Ergebnis, dass die Schaltfläche von dunkelgrau auf orange geändert wird:

Fortschritt & Lautstärke

Die letzte Anpassung des Erscheinungsbilds, die wir in diesem Dokument betrachten, besteht darin, die Farbe der Fortschritts- und Lautstärkeanzeigen zu ändern. Die Standardfarbe für beide ist Fuchsia, wie hier gezeigt:

Die Herausforderung besteht darin, die Klassennamen für die Elemente zu finden. Klicken Sie mit der gleichen Technik wie oben mit der rechten Maustaste auf ein Element, das Sie auswählen Element prüfen Wenn Sie zur gesamten Fortschrittsanzeige gelangen, sehen Sie die vjs-play-progress Klasse.

Im Chrome Elemente Panel, Maus über die div Element mit dem Klassennamen vjs-play-progress um Folgendes im Player zu sehen:

Ein Tool in Chrome ist hilfreich, wenn Sie nach dem Lautstärkeelement suchen. Beachten Sie, dass Sie die Lautstärke nur sehen, wenn Sie mit der Maus über die Lautstärketaste fahren (mit der Maus darüber fahren). Führen Sie die folgenden Schritte aus, damit der Lautstärkepegel bei der Überprüfung und Suche nach dem richtigen Element angezeigt wird:

  • Klicken Sie mit der rechten Maustaste auf die Lautstärketaste und überprüfen Sie sie
  • Drücke den Elementschaltfläche umschalten (unten blau hervorgehoben) oben rechts im Stile Sektion
  • Überprüf den :schweben Feld (rot hervorgehoben in der Stile Abschnitt), um die Maus zu simulieren, während Sie die Elemente untersuchen
  • Führen Sie einen Drilldown durch, bis Sie das Lautstärkeelement finden (rot hervorgehoben in der Elemente Sektion)

Jetzt die background-color von diesen beiden Elementen muss gestylt werden. Sie möchten, dass sie dieselbe Farbe wie die Schaltfläche haben. Ändern Sie daher Ihre Stile wie hier gezeigt:

  <style>
    #video_1 .vjs-big-play-button {
      opacity: .6;
    }
    #video_1 .vjs-big-play-button,
    #video_1 .vjs-play-progress,
    #video_1 .vjs-volume-level {
      background-color: #B37D5B;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Beachten Sie, dass wir jeder Klasse die ID des Spielers voranstellen div Element. Dies bietet Relevanz für die Klassenänderungen hinsichtlich der CSS-Spezifität. Darüber wird im nächsten Abschnitt gesprochen.

Wie Sie gesehen haben, besteht der Ansatz darin, den Klassennamen zu finden, der dem Element entspricht, das Sie ändern möchten, und ihn dann zu formatieren. Steuert Sichtbarkeit

Sie können festlegen, wie lange die Steuerelemente sichtbar sind. Sie verwenden die options().inactivityTimeout Eigenschaft, dies zu tun. Sie setzen den Wert auf eine Zahl, die die Sekunden darstellt, in denen die Steuerelemente nach dem Ende einer Aktivität auf dem Player sichtbar bleiben. Damit die Steuerelemente immer sichtbar bleiben, setzen Sie den Wert auf 0.

Der folgende Code zeigt die Einstellung der inactivityTimeout zu 0 Dadurch werden die Steuerelemente immer sichtbar.

<video-js id="myPlayerID"
  data-account="1752604059001"
  data-player="04ee9f58-9d08-43ea-803f-abda86776db2"
  data-embed="default"
  class="video-js" controls></video-js>
<script src="//players.brightcove.net/1752604059001/04ee9f58-9d08-43ea-803f-abda86776db2_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.options().inactivityTimeout = 0;
  });
</script>

Symbolreihenfolge der Steuerleiste

Sie können die Reihenfolge der Symbole in der Steuerleiste mithilfe von steuern order Stil eine Nummer zugewiesen. Beispielsweise, 5 platziert ein Symbol nach dem Fortschrittsbalken, aber vor der Lautstärketaste, und 9 Platziert das Symbol ganz rechts in der Steuerleiste. Für ältere Browser sind möglicherweise weitere Stile erforderlich, wie hier gezeigt:

{
  order: 5; /* after the progress bar but before the volume button */
  -webkit-box-ordinal-group: 5;
  -moz-box-ordinal-group: 5;
  -webkit-order: 5;
  -ms-flex-order: 5;
}

CSS-Spezifität

Die CSS-Spezifität bezieht sich auf die Art und Weise, wie Browser bestimmen, welche Eigenschaftswerte basierend auf ihrer Relevanz auf ein Element angewendet werden. Einzelheiten zu diesem Konzept finden Sie im Besonderheit Artikel.

Überprüfen Sie in Chrome die große Wiedergabetaste. Beachten Sie, dass die background Eigenschaft mit mehreren Klassen festgelegt wird. Dies bedeutet, dass wir nicht nur die eine .vjs-big-play-button Klasse verwenden können, um diese Eigenschaft zu überschreiben.

Hier sind drei Ansätze, um den großen Play-Button anzupassen:

  • Verweise auf eine dem Spieler zugewiesene ID.
  • Verwenden Sie mehrere Klassen, um die Relevanz der Eigenschaften zu erhöhen.
  • Verwenden Sie JavaScript, um das id-Attribut für den Player festzulegen und dann im CSS darauf zu verweisen.

Verweisen auf eine ID

Da ein id Attribut eine hohe Relevanz für die CSS-Spezifität aufweist, überschreibt das Hinzufügen eines id zum CSS-Selektor background-color eigentum.

Falls noch nicht geschehen, fügen Sie ein . hinzu id dem zuschreiben video Elemente wie folgt:

    <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

Fügen Sie im CSS den ID-Wert zum .vjs-big-play-button Klassenselektor hinzu. Überschreiben Sie die Hintergrundfarbe und die Deckkraft wie folgt:

  <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Du solltest sehen, dass der große Play-Button jetzt orange ist.

Verwenden mehrerer Klassen

Seit der background Die Eigenschaft für die große Wiedergabetaste wird mit zwei Klassen festgelegt. Sie benötigen mindestens zwei Klassen im CSS-Selektor, um diese Eigenschaft zu überschreiben.

Fügen Sie die .video-js Klasse zum .vjs-big-play-button Klassenselektor hinzu. Denken Sie daran, dass die .video-js Klasse auf dem Videocontainer festgelegt ist.

Mit diesen Werten sagen wir, wählen Sie alle Elemente mit dem Klassennamen von aus vjs-big-play-button Das sind Nachkommen von Elementen mit dem Klassennamen .video-js.

<style>
  .video-js .vjs-big-play-button {
    background-color: #B37D5B;
    opacity: .6;
  }
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

Du solltest sehen, dass der große Play-Button jetzt orange ist. Das Arbeiten mit mehreren Klassen kann schwierig sein und sich auf mehrere Elemente auswirken, während eine ID für ein Element spezifisch ist.

Verwenden von JavaScript

Sie können JavaScript auch verwenden, um dem großen Play-Button dynamisch eine ID hinzuzufügen. Dann können Sie diese ID aus dem CSS referenzieren.

Fügen Sie Code hinzu, um dem Element „Big Play Button“ eine ID hinzuzufügen.

  • Linie 173: Ruft einen Verweis auf das erste Element ab, das eine Klasse von hat vjs-big-play-button.
  • Linie 174: Fügt dem großen Play-Button ein id Attribut hinzu.
  <script type="text/JavaScript">

    var playButton = document.getElementsByClassName("vjs-big-play-button")[0];
    playButton.setAttribute("id", "myPlayButton");

  </script>

Ändern Sie im CSS den Selektor, um den myPlayButton ID-Namen zu verwenden.

  <style>
    #myPlayButton {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

Du solltest sehen, dass der große Play-Button jetzt orange ist.
