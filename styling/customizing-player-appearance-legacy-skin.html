    Anpassen des Spielerauftritts - Legacy

    Dieses Dokument enthält Informationen über den Brightcove Player 1.x-Skin (Graphit), nicht den aktuellsten Skin. Dieser Übergang zur neueren Haut fand in der zweiten Dezemberwoche 2015 statt. Die aktuellste Version des Dokuments finden Sie in diesem Dokument.

    Dieses Thema ist eine Einführung in das Anpassen des Erscheinungsbilds des neuen Players / styling / customizing-player-Erscheinungsbild.html. Der Inhalt verwendet die Entwicklungstools von Chrome, aber alle modernen Browser verfügen über eine gleichwertige Funktionalität. Eine kurze Einführung in die Verwendung von Entwicklungstools in verschiedenen Browsern finden Sie in der Grundlagen des JavaScript-Debuggens Dokument.

    Erstellen Sie eigene Skin

    Wenn Sie den Standard-Skin nicht verwenden und Ihre eigenen erstellen möchten, ist dies mit der Player Management API möglich. Das Player-Konfigurationshandbuch enthält die erforderlichen Informationen.

    Player

    Hinweis: Dieses Dokument setzt einige Kenntnisse in CSS voraus und verwendet die Entwicklungstools eines Browsers, um die Klassen von HTML-Elementen anzuzeigen und anwendbares CSS zu untersuchen. Wenn Sie nicht über die erforderlichen Grundkenntnisse verfügen, ist es am besten, wenn Sie die durcharbeiten Schnellstart: Spieleranpassung - Legacy Skin.

    Du kannst das Aussehen des Spielers ändern, aber du musst zuerst eine Möglichkeit haben, es anzugehen. Um dies zu sehen, durchsuchen Sie eine HTML-Seite mit dem eingefügten Player embedded_in_page-Code. In dem Elemente Suchen Sie im Abschnitt der Entwicklungstools den ersten <div> und Sie werden sehen, dass es eine Klasse mit dem Wert hat video-js unter anderem zugewiesen.

    Sobald du das weißt, kannst du Stile verwenden, um den Spieler selbst zu verändern. Um beispielsweise die Größe des Players zu ändern, können Sie diesen Stil verwenden:

    <style>
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

    iframe-Spieler

    Wenn Sie die Iframe-Implementierung des Players verwenden, ist die Situation anders. Sie sehen immer noch den Spieler mit dem video-js Klasse, aber es wird natürlich innerhalb des Iframes sein.

    Die von Ihnen erstellten Stile funktionieren weiterhin mit dem Player im iframe, es muss jedoch eine CSS-Datei erstellt und mit dem Player verknüpft werden, der Studio verwendet. Sie tun dies, indem Sie zum gehen Spieler Klicken Sie dann auf das Video, mit dem Sie das CSS verknüpfen möchten, und klicken Sie dann auf Plugins Abschnitt fügen Sie den Pfad Ihrer CSS-Datei hinzu.

    Wenn Sie den Iframe selbst anpassen möchten, können Sie dies tun, indem Sie a hinzufügen class Attribut auf die iframe Element (Zeile 27). In diesem Fall ist der Klassenwert videoFrame. Verwenden Sie dann ein <style> Tag, um wie gewünscht zu ändern. Im Beispiel unten die width und height geändert werden.

    <!doctype html>
<html>

<head>
  <meta charset="UTF-8">
  <title>Customize iframe</title>

  <!-- Page styles -->
  <style>
    .videoFrame {
      width: 640px;
      height: 360px;
    }
  </style>
</head>

<body>
  <iframe class="videoFrame" src='//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.html' allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
</body>

</html>

    Play-Button

    Wenn Sie das Aussehen des Play-Buttons ändern möchten, müssen Sie zuerst wissen, wie Sie ihn ansprechen. Stellen Sie in Chrome sicher, dass die Entwicklungswerkzeuge geöffnet sind und dann mit der rechten Maustaste auf die Schaltfläche klicken. Wählen Sie aus der angezeigten Auswahl Element prüfen.

    In dem Elemente Im Abschnitt der Entwicklungstools wird der HTML-Code angezeigt, der dem Schaltflächenelement entspricht.

    Aus dem oben hervorgehobenen Code sehen Sie, dass die der Schaltfläche zugewiesene Klasse ist vjs-big-play-button. Die Farbe der Wiedergabetaste (passend zum Tiger) und die Deckkraft werden mit diesen Stilen geändert:

      <style>
    .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    Sie würden denken, dass dieser Ansatz funktionieren würde, aber das tut er nicht. Wenn Sie mit CSS arbeiten, müssen Sie sich der Eigenschaftsspezifität bewusst sein. Dies bezieht sich darauf, wie der Browser bestimmt, welche Eigenschaftswerte basierend auf ihrer Relevanz auf ein Element angewendet werden.

    Eine Möglichkeit, die Schaltflächenänderungen anzuwenden, besteht darin, auf eine ID in der CSS-Auswahl zu verweisen. Dies liegt daran, dass IDs eine hohe CSS-Spezifität aufweisen. Weitere Einzelheiten finden Sie in der CSS-Spezifität Sektion

    Fügen Sie zunächst eine hinzu id Attribut auf die div Element für den Spielercode.

        <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

    <script src="//players.brightcove.net/1752604059001/68f30408-8fb4-431d-accb-8c5baa8c4790_default/index.min.js"></script>

    Referenzieren Sie dann den großen Play-Button des Spielers im CSS wie folgt:

      <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    Hier ist das Ergebnis der Änderung der Schaltfläche von dunkelgrau auf orange:

    Mittlere Wiedergabetaste

    Sie können die Wiedergabetaste sowohl horizontal als auch vertikal zentrieren, indem Sie die folgenden Stile verwenden:

    top: 0px;
left: 0px;
right: 0px;
bottom: 0px;
margin: auto;

    Play-Taste ausblenden

    Sie können die Wiedergabetaste vollständig ausblenden, indem Sie den folgenden Stil verwenden:

    display: none;

    Der folgende CodePen erlaubt es Ihnen, mit dem Play-Button zu experimentieren. Beachten Sie, dass das Dreieck in der Wiedergabetaste tatsächlich eine Schriftart ist und seine Größe mit einem font-size Stil gesteuert wird.

    Fortschritt & Volumen

    Die letzte Anpassung des Erscheinungsbilds, die wir in diesem Dokument betrachten, besteht darin, die Farbe der Fortschritts- und Lautstärkeanzeigen zu ändern. Die Standardfarbe für beide ist Fuchsia, wie hier gezeigt:

    Die Herausforderung besteht darin, die Klassennamen für die Elemente zu finden. Klicken Sie mit der gleichen Technik wie oben mit der rechten Maustaste auf ein Element, das Sie auswählen Element prüfen Wenn Sie zur gesamten Fortschrittsanzeige gelangen, sehen Sie die vjs-play-progress Klasse.

    Im Chrome Elemente Panel, Maus über die div Element mit dem Klassennamen vjs-play-progress um Folgendes im Player zu sehen:

    Ein Tool in Chrome ist hilfreich, wenn Sie nach dem Lautstärkeelement suchen. Beachten Sie, dass Sie die Lautstärke nur sehen, wenn Sie mit der Maus über die Lautstärketaste fahren (mit der Maus darüber fahren). Führen Sie die folgenden Schritte aus, um den Lautstärkepegel bei der Überprüfung und Suche nach dem richtigen Element anzuzeigen:

    • Klicken Sie mit der rechten Maustaste auf die Lautstärketaste und überprüfen Sie sie
    • Drücke den Elementschaltfläche umschalten (unten blau hervorgehoben) oben rechts im Stile Sektion
    • Überprüf den :schweben Feld (rot hervorgehoben in der Stile Abschnitt), um die Maus zu simulieren, während Sie die Elemente untersuchen
    • Führen Sie einen Drilldown durch, bis Sie das Lautstärkeelement finden (rot hervorgehoben in der Elemente Sektion)

    Jetzt die background-color von diesen beiden Elementen muss gestylt werden. Sie möchten, dass sie dieselbe Farbe wie die Schaltfläche haben. Ändern Sie daher Ihre Stile wie hier gezeigt:

      <style>
    #video_1 .vjs-big-play-button {
      opacity: .6;
    }
    #video_1 .vjs-big-play-button,
    #video_1 .vjs-play-progress,
    #video_1 .vjs-volume-level {
      background-color: #B37D5B;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    Beachten Sie, dass wir jeder Klasse die ID des Spielers voranstellen div Element. Dies bietet Relevanz für die Klassenänderungen hinsichtlich der CSS-Spezifität. Darüber wird im nächsten Abschnitt gesprochen.

    Die Ergebnisse sehen wie folgt aus:

    Mit dieser Grundierung erhalten Sie einen kurzen Überblick über das Styling des Players. Wie Sie gesehen haben, besteht der Ansatz darin, den Klassennamen zu finden, der dem Element entspricht, das Sie ändern möchten, und ihn dann zu formatieren. Denken Sie daran, wie in der erwähnt iframe Wenn Sie den Iframe-Player verwenden, müssen Sie die Stile in eine Datei einfügen und dann eine Curl-Anweisung verwenden, um die Stile dem Player zuzuordnen.

    Denken Sie daran, dass es eine bewährte Methode ist, die Anpassungen zu Ihrer Player-Konfiguration hinzuzufügen, damit sie optimiert werden können und mit Ihrem Player überall dort eingesetzt werden können, wo er eingebettet ist.

    Steuert die Sicht

    Sie können festlegen, wie lange die Steuerelemente sichtbar sind. Sie verwenden die options().inactivityTimeout Eigenschaft, dies zu tun. Sie setzen den Wert auf eine Zahl, die die Sekunden darstellt, in denen die Steuerelemente nach dem Ende einer Aktivität auf dem Player sichtbar bleiben. Damit die Steuerelemente immer sichtbar bleiben, setzen Sie den Wert auf 0.

    Der folgende Code zeigt die Einstellung der inactivityTimeout zu 0 Dadurch werden die Steuerelemente immer sichtbar.

    <video-js id="myPlayerID"
  data-account="1752604059001"
  data-player="04ee9f58-9d08-43ea-803f-abda86776db2"
  data-embed="default"
  class="video-js" controls></video-js>
<script src="//players.brightcove.net/1752604059001/04ee9f58-9d08-43ea-803f-abda86776db2_default/index.min.js"></script>

<script>
  videojs.getPlayer('myPlayerID').ready(function() {
    var myPlayer = this;
    myPlayer.options().inactivityTimeout = 0;
  });
</script>

    Symbolreihenfolge der Steuerleiste

    Sie können die Reihenfolge der Symbole in der Steuerleiste mithilfe von steuern order Stil eine Nummer zugewiesen. Zum Beispiel, 5 platziert ein Symbol nach dem Fortschrittsbalken, aber vor der Lautstärketaste, und 9 Platziert das Symbol ganz rechts in der Steuerleiste. Für ältere Browser sind möglicherweise weitere Stile erforderlich, wie hier gezeigt:

    {
  order: 5; /* after the progress bar but before the volume button */
  -webkit-box-ordinal-group: 5;
  -moz-box-ordinal-group: 5;
  -webkit-order: 5;
  -ms-flex-order: 5;
}

    Ein Beispiel für das Hinzufügen eines Symbols (in diesem Fall zum Herunterladen einer Datei) wird im Brightcove Player-Beispiel gezeigt: Laden Sie Video Plugin-Dokument herunter

    CSS-Spezifität

    Die CSS-Spezifität bezieht sich auf die Art und Weise, wie Browser anhand ihrer Relevanz bestimmen, welche Eigenschaftswerte auf ein Element angewendet werden. Einzelheiten zu diesem Konzept finden Sie im Artikel Specificity .

    Untersuchen Sie in Chrome den großen Play-Button. Beachten Sie, dass die background Eigenschaft mit mehreren Klassen festgelegt wird. Dies bedeutet, dass wir nicht nur die eine .vjs-big-play-button Klasse verwenden können, um diese Eigenschaft zu überschreiben.

    Hier sind drei Ansätze, um den großen Play-Button anzupassen:

    • Verweise auf eine dem Spieler zugewiesene ID.
    • Verwenden Sie mehrere Klassen, um die Relevanz der Eigenschaften zu erhöhen.
    • Verwenden Sie JavaScript, um das ID-Attribut für den Player festzulegen und dann im CSS darauf zu verweisen.

    Referenzieren einer ID

    Da ein id Attribut eine hohe Relevanz in Bezug auf die CSS-Spezifität hat, überschreibt das Hinzufügen eines id zum CSS-Selektor die background-color Eigenschaft.

    Wenn Sie dies noch nicht getan haben, fügen Sie eine hinzu id Attribut auf die video Element wie folgt:

        <video-js id="video_1"
      data-account="1752604059001"
      data-player="68f30408-8fb4-431d-accb-8c5baa8c4790"
      data-embed="default"
      class="video-js" controls></video-js>

    Fügen Sie im CSS den ID-Wert zum .vjs-big-play-button Klassenselektor hinzu. Überschreiben Sie die Hintergrundfarbe und die Deckkraft wie folgt:

      <style>
    #video_1 .vjs-big-play-button {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    Du solltest sehen, dass der große Play-Button jetzt orange ist.

    Verwenden mehrerer Klassen

    Seit der background Die Eigenschaft für die große Wiedergabetaste wird mit zwei Klassen festgelegt. Sie benötigen mindestens zwei Klassen im CSS-Selektor, um diese Eigenschaft zu überschreiben.

    Fügen Sie die .video-js Klasse der .vjs-big-play-button Klassenauswahl hinzu. Denken Sie daran, dass die .video-js Klasse im Videocontainer festgelegt ist.

    Mit diesen Werten sagen wir, wählen Sie alle Elemente mit dem Klassennamen aus vjs-big-play-button Das sind Nachkommen von Elementen mit dem Klassennamen .video-js.

    <style>
  .video-js .vjs-big-play-button {
    background-color: #B37D5B;
    opacity: .6;
  }
  .video-js {
    width: 640px;
    height: 360px;
  }
</style>

    Du solltest sehen, dass der große Play-Button jetzt orange ist. Die Arbeit mit mehreren Klassen kann schwierig sein und sich auf mehrere Elemente auswirken, während eine ID für ein Element spezifisch ist.

    Verwendung von JavaScript

    Sie können auch JavaScript verwenden, um dem großen Play-Button dynamisch eine ID hinzuzufügen. Dann können Sie diese ID aus dem CSS beziehen.

    Fügen Sie Code hinzu, um dem Big-Play-Button-Element eine ID hinzuzufügen.

    • Zeile 173: Ruft einen Verweis auf das erste Element mit einer Klasse von vjs-big-play-buttonab.
    • Zeile 174: Fügt dem großen Play-Button ein id Attribut hinzu.
      <script type="text/JavaScript">

    var playButton = document.getElementsByClassName("vjs-big-play-button")[0];
    playButton.setAttribute("id", "myPlayButton");

  </script>

    Ändern Sie im CSS den Selektor, um den myPlayButton ID-Namen zu verwenden.

      <style>
    #myPlayButton {
      background-color: #B37D5B;
      opacity: .6;
    }
    .video-js {
      width: 640px;
      height: 360px;
    }
  </style>

    Du solltest sehen, dass der große Play-Button jetzt orange ist.
