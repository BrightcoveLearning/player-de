Schnellstart: Spieleranpassung - Legacy Skin

Dieses Dokument enthält Informationen zur Brightcove Player 1.x-Skin (Graphit), nicht zur aktuellsten Skin. Dieser Übergang zur neueren Haut fand in der zweiten Dezemberwoche 2015 statt. Die aktuellste Version des Dokuments finden Sie in diesem Dokument.

In diesem Thema werden praktische Schritte bereitgestellt, mit denen das Erscheinungsbild des Brightcove-Players angepasst wird. Um das Erscheinungsbild des Brightcove-Players anzupassen, müssen Sie Cascading Style Sheets (CSS) verwenden. In diesem Dokument werden das Erscheinungsbild und die Position der Wiedergabetaste geändert, mit der das Video anfänglich überlagert wird.

Überblick

Dieses Dokument enthält mehrere Abschnitte, die CSS mit der In-Page-Einbettung verwenden, und die Iframe-Implementierungen des Players sind sehr unterschiedlich.

Der Inhalt in diesem Dokument verwendet die Entwicklungstools von Chrome, aber alle modernen Browser verfügen über eine gleichwertige Funktionalität. Eine kurze Einführung in die Verwendung von Entwicklungstools in verschiedenen Browsern finden Sie im Debugging-Grundlagen dokumentieren.

Hinweis: Ein grundlegendes Player-Styling wird in Video Cloud Studio bereitgestellt. Weitere Informationen finden Sie unter Spieler stylen.

Bestimmen Sie die Spielerauswahl

Mit CSS-Selektoren können Sie HTML-Elemente auf einer Seite auswählen und bearbeiten. CSS-Selektoren werden verwendet, um HTML-Elemente basierend auf ihrer ID, Klasse, ihrem Typ, ihrem Attribut und mehr zu „suchen“ (oder auszuwählen). Das erste, was Sie tun müssen, wenn Sie versuchen, das Aussehen eines Elements auf einer HTML-Seite zu ändern, ist, den Selektor zu bestimmen, der zum Ansprechen des Elements verwendet werden soll. Die folgenden Schritte zeigen, wie Sie den Selektor für den Spieler finden und dann die Größe des Spielers ändern.

  1. Öffnen Sie in Video Cloud Studio das Medienmodul und wählen Sie ein zu veröffentlichendes Video aus.
  2. Kopiere das Auf Seite Code veröffentlichen.
  3. Erstellen Sie mit Ihrem HTML-Editor eine neue HTML-Seite.
  4. Fügen Sie den Veröffentlichungscode in den Hauptteil Ihrer HTML-Seite ein.
  5. Durchsuchen Sie die neu erstellte Seite. Standardmäßig wird der Player mit Standardabmessungen von 300 x 150 Pixel dimensioniert.
  6. Öffnen Sie die Entwicklungstools Ihres Browsers und führen Sie einen Drilldown in den HTML-Code durch, um die zu finden div Tag, der den Player umgibt. Sie werden das sehen div Tag hat eine Klasse von video-js. Das div das enthält den Spieler wird immer eine Klasse von haben video-js . video-js ist der Klassenname, den wir für das Styling des Players benötigen.
  7. Sehen Sie sich den Screenshot unten an, um die rot hervorgehobene Klasse zu sehen. Grün hervorgehoben ist das Video-Tag Brightcove Player.

    Hinweis: Je nach verwendetem Betriebssystem und verwendetem Browser wird möglicherweise die Flash-Fallback-Version des Players angezeigt. Wenn Ihr Player die Flash-Fallback-Version des Players verwendet, wird a nicht angezeigt video Etikett. Stattdessen sehen Sie die div um ein objectTag, wie unten hervorgehoben.

  8. Nachdem Sie die richtige Klasse für einen CSS-Selektor kennen, können Sie mit diesem Klassenselektor die Größe des Players ändern. Das style Block unten sollte direkt über dem Ende des platziert werden head Abschnitt Ihrer HTML-Seite. In der CSS-Terminologie ist dies ein Beispiel für die Verwendung von a Klassenselektor. 
            <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
        </style>
      </head>
  9. Speichern und durchsuchen Sie Ihre HTML-Seite und Sie werden sehen, dass der Player eine neue Größe hat.

Button-Selektor abspielen

Ein anderer Ansatz wird verwendet, um den Selektor für den Play-Button zu finden. Die folgenden Schritte zeigen Ihnen, wie Sie den richtigen Selektor für den Play-Button finden und dann dessen Aussehen ändern.

  1. Durchsuchen Sie Ihre HTML-Seite.
  2. Klicken Sie mit der rechten Maustaste auf die Wiedergabetaste, die den Player überlagert, und wählen Sie Element prüfen.
  3. Notiere dass der div Das markierte Tag stellt die Wiedergabetaste dar und hat eine Klasse von vjs-big-play-button.
  4. Ändern Sie Ihre Stile so, dass sie CSS enthalten, das die Farbe und Deckkraft der Wiedergabetaste ändert (Zeilen 17-18) und die Schaltfläche im Player zentriert (Zeilen 19-23). 
          <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
          .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
        </style>
  5. Durchsuchen Sie Ihre Seite und Sie werden sehen, dass nicht alle Stile verwendet wurden. Der Knopf ist nicht rot.
  6. Schauen Sie rechts von Ihren Entwicklertools nach und Sie sehen eine Zusammenfassung Ihrer Stile. Sie können nach unten schauen und sehen, dass das Ändern des Hintergrunds in Rot überschrieben wird und dadurch dargestellt wird, dass er durchgestrichen wird.
    Hinweis: In Safari wird der Stil möglicherweise nicht durchgestrichen, obwohl er außer Kraft gesetzt wird.

    Was passiert ist, ist, dass zahlreiche Änderungen an der Hintergrundfarbe vorgenommen werden. Der von Ihnen festgelegte Stil wurde im „kaskadierenden“ Teil von CSS überschrieben. Sie können dies ändern, indem Sie die Funktionalität von CSS verwenden, die als Spezifität bezeichnet wird. Die Spezifität bestimmt, welche CSS-Regel von Browsern angewendet wird und ist eine Art von Gewichtung, die sich darauf auswirkt, wie Ihre Cascading Stylesheet-Regeln (CSS) angezeigt werden. Mit Spezifität können Sie Ihren Stil wichtiger machen als andere und zwingen, ihn zu verwenden. Es gibt eine kurze Einführung in die CSS-Spezifität in der Passen Sie das Aussehen des Spielers an - Legacy Skin doc und zahlreiche Referenzen in jeder Suche.

  7. Notiere dass der div Tag umgibt den Player mit zahlreichen Klassen verbunden.
  8. In diesem Fall mit der Kombination von video-js und vjs-mouse wird Ihre Stile "verbessern", um Vorrang zu haben. Ändern Sie Ihre Stile, um den neuen Kombinationsselektor zu verwenden. Beachten Sie, dass zwischen den ersten beiden Klassen kein Leerzeichen steht, aber vor der letzten Klasse ein Leerzeichen (Einzelheiten dazu finden Sie in den Regeln für die Spezifität). 
              .video-js.vjs-mouse .vjs-big-play-button {
            background-color: #ff0000;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
  9. Durchsuchen Sie Ihre Seite und sehen Sie, dass alle Stile jetzt angewendet werden.
  10. Schauen Sie erneut rechts von Ihren Entwicklertools nach und Sie sehen eine Zusammenfassung Ihrer Stile. Jetzt sehen Sie, dass Ihre Hintergrundfarbe nicht überschrieben wird.

In-Page-Einbetten

Bei der Arbeit mit den oben genannten Selektoren und Stilen wurden die Ergebnisse mit der in die Seite eingebetteten Implementierung des Players verwendet, wie hier gezeigt:

      <!doctype html>
      <html>
      
      <head>
          <meta charset="UTF-8">
          <title>Embed in page tester</title>
         <style>
          .video-js {
            height: 344px;
            width: 610px;
          }
          .video-js.vjs-mouse .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
         </style>
      </head>
      <body>
      
        <video
          data-video-id="4443311217001"
          data-account="1507807800001"
          data-player="30a9c9e0-295b-4102-a7af-a3f86bba91ab"
          data-embed="default"
          class="video-js" controls></video>
        <script src="//players.brightcove.net/1507807800001/30a9c9e0-295b-4102-a7af-a3f86bba91ab_default/index.min.js"></script>
      
      
      </body>
      </html>

Dies ist eine sehr gebräuchliche Methode, CSS mit einer HTML-Seite zu verwenden. Sie sehen das style Block ist auf der Seite mit dem video Etikett.

Hinweis: Abhängig von Ihrer CSS-Erfahrung fragen Sie sich möglicherweise, warum der obige Code keine zugewiesen hat id Attribut auf die video Etikett. Sie könnten dann einen Selektor wie verwenden #myPlayer .vjs-big-play-button. In der Tat hättest du es tun können und es hätte funktioniert. Das Problem ist, dass das mit ID-Selektoren erstellte CSS bei Verwendung der Iframe-Player-Implementierung nicht funktioniert, sodass nur vorhandene Klassennamen verwendet wurden. Indem Sie keine verwenden id -Attributs sind die Stile direkt auf Iframe-Player-Implementierungen anwendbar.

iframe

Es überrascht nicht, dass die Iframe-Implementierung den Brightcove-Player in einem Iframe platziert, den Sie dann auf einer HTML-Seite platzieren. Sie können den eigentlichen Iframe selbst stylen, beispielsweise seine Größe ändern, indem Sie Stile auf der Seite verwenden, die den Iframe enthält, aber Stile auf der Seite des Iframes kaskadieren NICHT in den Iframe selbst. Der Iframe zieht buchstäblich eine separate HTML-Seite ein, daher müssen die gewünschten Stile für den Inhalt des Iframe mit der HTML-Seite im Iframe verknüpft sein.

Wenn Sie die iframe-Implementierung des Brightcove Players verwenden, lauten die Schritte zum Anwenden von CSS:

  • Platzieren Sie Stile in einer Datei für das Internet zugänglich
  • Verknüpfen Sie das Stylesheet mit dem Player (erledigt mit Studio oder der Player Management API)
Hinweis: Die folgenden Schritte gehen davon aus, dass Sie mit dem HTML-Code arbeiten, den Sie in den vorherigen Schritten ausgeführt und im obigen Codeausschnitt angezeigt haben. Wenn Sie das als funktionierendes Beispiel behalten möchten, kopieren Sie einfach den Code in eine andere Datei und starten Sie die folgenden Schritte in der neuen Datei.
  1. Veröffentlichen Sie in Studio ein Video und kopieren Sie den Code für eine Iframe-Implementierung des Players.
  2. Ersetzen Sie auf Ihrer HTML-Seite den In-Page-Einbettungscode durch den Iframe-Code.
  3. Ändere das .video-js Klassenauswahl zu einer Elementauswahl mit iframe als das Element. Verlassen width und height das gleiche, wie hier gezeigt: 
      iframe {
      height: 344px;
      width: 610px;
    }
  4. Vergewissern Sie sich, dass Ihre HTML-Seite wie folgt aussieht: 
          <!doctype html>
      <html>
      <head>
        <meta charset="utf-8">
        <title>iframe tester</title>
      
         <style>
          iframe {
            height: 344px;
            width: 610px;
          }
          .video-js.vjs-mouse .vjs-big-play-button {
            background-color: red;
            opacity: .3;
            top: 0px;
            left: 0px;
            right: 0px;
            bottom: 0px;
            margin: auto;
          }
         </style>
      <body>
      
        <iframe src='//players.brightcove.net/1507807800001/30a9c9e0-295b-4102-a7af-a3f86bba91ab_default/index.html?videoId=4443311217001'
          allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
      
      </body>
      </html>
  5. Durchsuchen Sie die Seite und Sie werden sehen, dass der Iframe-Player die richtige Größe hat. Wie bereits erwähnt, funktionieren die Stile für die Schaltfläche NICHT.
  6. Erstellen Sie eine Datei mit dem Namen customize-qs.css und speichern Sie sie an einem auf das Internet zugänglichen Speicherort.
  7. Schneiden Sie die Stile für die Schaltfläche aus und fügen Sie sie in die Datei ein und speichern Sie sie. Schneiden Sie nur die Stile für die Schaltfläche aus, Sie können die nicht einschließen style Stichworte. Die Datei sollte nur diesen Inhalt enthalten: 
          .video-js.vjs-mouse .vjs-big-play-button {
        background-color: red;
        opacity: .3;
        top: 0px;
        left: 0px;
        right: 0px;
        bottom: 0px;
        margin: auto;
      }
  8. Vergewissern Sie sich, dass Ihre HTML-Seite wie folgt aussieht: 
          <!doctype html>
      <html>
      <head>
        <meta charset="utf-8">
        <title>iframe tester</title>
      
        <style>
          iframe {
            height: 344px;
            width: 610px;
          }
        </style>
      <body>
      
        <iframe src='//players.brightcove.net/1507807800001/30a9c9e0-295b-4102-a7af-a3f86bba91ab_default/index.html?videoId=4443311217001'
          allowfullscreen webkitallowfullscreen mozallowfullscreen></iframe>
      
      </body>
      </html>
  9. Wechseln Sie in Studio in das Player-Modul und bearbeiten Sie den Player, den Sie für diesen Schnellstart verwenden.
  10. Suchen Sie die PLUGINS Abschnitt und klicken Sie auf die Bearbeiten Taste.
  11. Drücke den CSS Geben Sie den Pfad zu Ihrer CSS-Datei ein und geben Sie ihn wie folgt ein:
  12. Klicken speichern um Ihre Änderungen in der zu speichern PLUGINS Sektion.
    Der Vorschauplayer oben auf der Seite wird aktualisiert, um die auf die Schaltfläche angewendeten CSS-Stile anzuzeigen.
  13. Klicken Sie oben auf der Seite auf Veröffentlichen um deinen Player zu veröffentlichen.
  14. Kehren Sie zu Ihrem Browser zurück und aktualisieren Sie die Seite. Sobald die veröffentlichten Änderungen aktiv sind, sehen Sie, dass Ihre Stile auf den Iframe-Player angewendet werden. Es kann einige Minuten dauern, bis die veröffentlichten Änderungen sichtbar sind.

Stile erkunden

In diesem Abschnitt wird ein JSBin verwendet, mit dem Sie Stile erkunden und mehr Anpassungen an der großen Wiedergabetaste vornehmen können als oben gezeigt. Sobald Sie einen Look erhalten haben, können Sie das generierte CSS einfach kopieren und wie oben gezeigt mit beiden Player-Implementierungen verwenden.

Ein Problem, das verwirrend erscheinen mag, ist, dass der eigentliche Pfeil in der Schaltfläche kein Bild, sondern Teil einer Schriftfamilie ist. Aus diesem Grund steuern Sie seine Größe mit font-size und verwenden ems sie als Einheiten.
