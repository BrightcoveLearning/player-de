    Playlist UI Plugin Changelog

    Im Folgenden werden Änderungen am Plugin verfolgt:

    Changelog

    7. Mai 2020

    v3.8.0

    • Option zum Anzeigen von Videobeschreibungen in Miniaturansichten von Wiedergabelisten aktiviert

    10. Februar 2020

    v3.7.0

    • Arabische Übersetzungen hinzugefügt
    • Es wurde ein Fehler behoben, der während der Initialisierung bei Verwendung der nächsten Überlagerung ausgelöst wurde

    9. Okt 2019

    v3.6.3

    • Zeigen Sie im Fokus die Registerkarte Wiedergabeliste an

    8. Oktober 2019

    v3.6.2

    • Ein nachfolgendes UI-Problem mit horizontalen Wiedergabelisten in Firefox und IE11 wurde behoben

    9 Sep 2019

    v3.6.1

    • Es wurde ein Problem behoben, bei dem horizontale Wiedergabelisten in IE11 und Firefox nicht korrekt angezeigt wurden

    27 August 2019

    v3.6.0

    • Übersetzungen für Chinesisch, Deutsch, Spanisch, Französisch, Japanisch und Koreanisch hinzugefügt

    16. Juli 2019

    v3.5.0

    • Fügt Unterstützung für die iframe-ähnliche Wiedergabelisten-Benutzeroberfläche für In-Page-Player über ein Containerelement hinzu

    20 Mrz 2019

    v3.4.4

    • Aktualisieren videojs-playlist bis 4.3.1; Dadurch werden die in der Videojs-Wiedergabeliste 4.3.0 eingeführten Regressionen behoben

    14. Januar 2019

    v3.4.3

    • Aktualisiert videojs-playlist bis 4.3.0; Dies behebt ein ungewöhnliches Problem mit Wiedergabelisten, die mehrere Elemente mit derselben Quell-URL enthalten

    29. Oktober 2018

    v3.4.2

    • Hinzugefügt playlistPicker Option, standardmäßig auf true , um das Entfernen der Wiedergabelistenauswahl zu ermöglichen (wenn übergeben als false)
    • Bugfix: Erstellen Sie die horizontale Wiedergabeliste in Iframe-Playern mit einer statischen Höhe von 20% anstelle einer variablen Höhe
    • Bugfix: Behebung eines Problems, bei dem eine vertikale Bildlaufleiste in horizontalen Wiedergabelisten angezeigt wurde
    • Bugfix: Entfernen Sie die Playlist-Benutzeroberfläche, wenn der Player entsorgt ist

    3 Okt 2018

    v3.4.1

    • Es wurde ein Problem behoben, durch das eine vertikale Bildlaufleiste in horizontalen Wiedergabelisten angezeigt wurde
    • Entfernen Sie die Playlist-Benutzeroberfläche, wenn der Player entsorgt ist

    25. September 2018

    v3.4.0

    • PlaylistPicker-Option hinzugefügt, standardmäßig true, um das Entfernen der Playlist-Auswahl zu ermöglichen.

    17. Sep 2018

    v3.3.2

    • Es wurde ein Problem behoben, durch das ein zur falschen Zeit geladener Wiedergabelisten-Player dazu führen konnte, dass die Benutzeroberfläche nicht gerendert wurde
    • Das Postinstall-Skript wurde entfernt, um Installationsprobleme zu vermeiden
    • Zum Schweigen gebracht a play Versprechen Ausnahme in Wiedergabelisten
    • Aktualisiert mit Plugin-Generator 7.2.0

    13. Juni 2018

    v3.3.1

    • Es wurde ein Problem behoben, bei dem die Schaltfläche für das nächste Element keinen Zeigercursor hatte

    23. Mai 2018

    v3.3.0

    • Unterstützung für die explizite Zuordnung eines Wiedergabelistenelements zu einem Player über Attribute

    9. Mai 2018

    v3.2.1

    • Es wurde ein Problem behoben, durch das ein Einbettungscode, der ein Skript vor einem Wiedergabelistencontainer auf einer Seite mit mehreren Spielern mit Wiedergabelisten platziert, dazu führen konnte, dass diese Wiedergabelisten nicht mehr in Ordnung waren

    9. April 2018

    v3.2.0

    • Shuffle-Option hinzugefügt, um die Wiedergabeliste bei jeder Änderung nach dem Zufallsprinzip zu sortieren

    29. Mrz 2018

    v3.1.1

    • Schneiden Sie längere Videotitel mit Ellipsen ab, wenn sie überlaufen

    25. Januar 2018

    v3.1.0

    • Aktualisieren zu videojs-playlist 4.2.0, Hinzufügen duringplaylistchange Veranstaltung und rest Option für die shuffle() Methode

    17. Jan 2018

    v3.0.1

    • Behobener Fall, in dem die Wiedergabeliste leer war, die Seitenleiste der Wiedergabeliste in einem Iframe jedoch weiterhin angezeigt wurde
    • Es wurde ein Problem behoben, bei dem ein automatischer Vorlauf auftrat, selbst wenn der Benutzer die Wiedergabe nach einem beendeten Ereignis neu startete
    • Das Klicken / Tippen auf Symbole in der nächsten Überlagerung und auf dem Endbildschirm auf Mobilgeräten wurde behoben

    22. Dez 2017

    v3.0.0

    • Features
      • Optional hinzufügen Als nächstes verschließbare Overlay-Karte
      • Optional hinzufügen Als nächstes Endbildschirm modal
      • Bessere Unterstützung für mehrere In-Page-Spieler durch intelligentere Suche nach dem zugehörigen Playlist-Element eines Spielers
      • Bundle Videojs-Wiedergabeliste als direkte Abhängigkeit
      • Unterstützen Sie horizontale Wiedergabelisten in iFrame-
    • Code Refactoring
      • Verschieben Sie die Seitenleiste in das Komponenten- und Refactor-Plugin, um
      • Refactor neu Als nächstes Merkmal
      • Tooling auf Plugin Generator v5.1.1 aktualisieren
    • ÄNDERUNGEN BRECHEN
      • Entfernt die Unterstützung für IE8
      • Unterstützt video.js/BrightCove Player v5 nicht mehr

    21. September 2017

    v2.1.2

    • Keine Änderungen

    25. Juli 2017

    v2.1.1

    • Keine Änderungen

    26. April 2017

    v2.1.0

    • Features
      • Fügen Sie eine optionale verschließbare Overlay-Karte "Up Next" hinzu
      • Fügen Sie optional das Endscreen-Modal "Up Next" hinzu
      • Bessere Unterstützung für mehrere In-Page-Spieler durch intelligentere Suche nach dem zugehörigen Playlist-Element eines Spielers
      • Bundle Videojs-Wiedergabeliste als direkte Abhängigkeit
      • Unterstützen Sie horizontale Wiedergabelisten in iFrame-
    • Code Refactoring
      • Verschieben Sie die Seitenleiste in das Komponenten- und Refactor-Plugin, um
      • Refactor neue 'Up Next'-Funktion
      • Tooling auf Plugin Generator v5.1.1 aktualisieren
    • ÄNDERUNGEN BRECHEN
      • Entfernt die Unterstützung für IE8
      • Unterstützt video.js/BrightCove Player v5 nicht mehr

    8 Sep 2016

    v2.0.0

    • Überarbeitete Benutzeroberfläche
      • Allgemeine Verbesserungen des Erscheinungsbilds
      • Größere Thumbnails
      • Platz für längere Videotitel
      • Wiedergabelisten-Schublade in Iframe-Einbettungen
      • Reaktionsschnelleres Layout

    6 Sep 2016

    v1.0.1

    • Behebung des Fehlers beim Einbetten der Wiedergabeliste in die Einbettung in die Seite

    16. August 2016

    v1.0.0

    • Die Benutzeroberfläche der Wiedergabeliste wurde in ein neues Plugin verschoben
