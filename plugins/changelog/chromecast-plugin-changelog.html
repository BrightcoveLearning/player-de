Chromecast Plugin Changelog

Die folgenden Tracks ändern sich am Plugin:

Changelog

8. Sep 2020

v2.0.0

  • Chromecast-Empfänger mit CAF-API neu implementiert

21. Mai 2020

v1.3.4

  • SSAI- und Katalogintegration für Standardplayer

7. Mai 2020

v1.3.3

  • Versuchen Sie den Player init bei einem Fehler erneut und sperren Sie die Standardplayer-Version

v1.3.2

v1.3.1

v1.3.0

  • Arabische Übersetzungen hinzugefügt
  • Bugfix: Verhindern Sie Append-Pufferfehler, indem Sie die maximale Zielpufferlänge auf 30 festlegen
  • Zeigen Sie den gesamten Gerätenamen anstelle des ersten Buchstabens an
  • Readme mit neuen Testverfahren aktualisiert
  • Die README wurde mit Schritten für unsichere Inhaltstests aktualisiert

v1.2.0

v1.1.5

19. Apr 2019

v1.1.4

  • Versuchen Sie nicht, in nicht sicheren Kontexten zu laufen, protokollieren Sie eine Warnung, wenn das Plugin nicht initialisiert werden kann

v1.1.2

  • Die API-Dokumente wurden aktualisiert

v1.1.1

  • Es wurde ein Syntaxfehler auf dem Receiver behoben, wodurch er nicht geladen wurde
