Verwendung in HTML

Wenn Sie die URLs für die Plugins direkt in einer HTML-Seite verwenden, verwenden Sie a <script> Tag für das JavaScript und a <link> Tag für das CSS. Beispiele folgen:

<script src="https://players.brightcove.net/videojs-vr/1/videojs-vr.min.js"></script>
<link href="https://players.brightcove.net/videojs-vr/1/videojs-vr.css" rel="stylesheet">
